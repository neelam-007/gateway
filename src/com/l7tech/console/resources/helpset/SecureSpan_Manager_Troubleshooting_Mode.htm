SecureSpan™ Manager Troubleshooting Mode

 NOTE: The SecureSpan™ Manager GUI (general user interface) will automatically open when you initiate Troubleshooting Mode. You must login to the SecureSpan™ Gateway that you want to troubleshoot in order to populate the MS DOS window.

The SecureSpan™ Manager interface contains two modes:

  • A GUI (general user interface) for all configuration and management tasks (see SecureSpan™ Manager Interfaces), and

  • A Troubleshooting Mode used to evaluate SecureSpan™ Gateway connectivity issues and messages.

Each mode is launched from a different shortcut icon in the program menu. To access the Troubleshooting Mode:

 

  1. From the Windows Desktop, click [Start] > All Programs > SecureSpan Manager > SecureSpan Manager in Troubleshooting Mode.

 NOTE: If the SecureSpan™ Manager was already open, then another instance of the program will launch when you initiate Troubleshooting Mode. The new instance is tied to the MS DOS window and will automatically close when the MS DOS window is closed.

  1. A blank MS DOS window appears, the Manager GUI launches, and the Login form appears. In the Login form:

    • Enter your user name and password into the respective User Name and Password fields

    • Select the target Gateway from the Gateway URL drop-down list. Click [OK]. A pop-up status message appears and then automatically disappears when the Gateway connection is established.

 NOTE: The MS DOS window may have become minimized during the SecureSpan™ Manager login process.

  1. The MS DOS window is populated with Gateway-specific information. Click the [X] in the top right corner to close the window.

 NOTE: The instance of the SecureSpan™ Manager GUI that was launched by the Troubleshooting Mode will automatically close when you close the MS DOS window.
