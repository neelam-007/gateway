SecureSpan™ Agent Interface

The SecureSpan™ Agent runs discretely in the background of a SecureSpan™ Solution implementation. The Agent interface is used for configuring Agent settings or for troubleshooting SecureSpan™ Gateway connectivity issues.

Interface Modes

The SecureSpan™ Agent interface contains two modes:

  • A GUI Mode used to create, configure, and delete SecureSpan™ Gateway connections, and

  • A Troubleshooting Mode used to evaluate Gateway connectivity issues and messages.

Each mode is launched from a different shortcut icon in the program menu.

GUI Mode

The SecureSpan™ Agent GUI Mode allows you to:

  • Register a new SecureSpan™ Gateway

  • Configure, view, edit, and delete settings for one or more Gateways

  • View Gateway connection messages.

To launch the Agent GUI Mode from the Windows Desktop, click [Start] > All Programs > SecureSpan Agent > Start SecureSpan Agent. The GUI interface contains three sections:

Main Menu:

Contains drop-down menus that include all of the program features, functions, and information.

Main Tool Bar:

Contains shortcut buttons to commonly-used program features.

Gateway List:

The main interface section contains a list of all SecureSpan™ Gateways registered in the Agent.

Troubleshooting Mode

NOTE: If not already open, the SecureSpan™ Agent GUI will automatically launch when you initiate Troubleshooting Mode.

The SecureSpan™ Agent Troubleshooting Mode launches an MS DOS window that displays detailed information about the messages communicated to the Agent by the SecureSpan™ Gateway. To launch the SecureSpan™ Agent Troubleshooting Mode from the Windows Desktop, click [Start] > All Programs > SecureSpan Agent > Start SecureSpan Agent in Troubleshooting Mode. To close the MS DOS window, click the [X] in the top right corner of the window.

 

NOTE: If you launched the GUI of the SecureSpan™ Agent by initiating Troubleshooting Mode, then the GUI will automatically close when you close the MS DOS window.
