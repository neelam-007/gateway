SecureSpan™ Agent Overview

NOTE: Only an overview of the SecureSpan™ Agent functionality and interface is provided in this Help System.

The SecureSpan™ Agent is an optional SecureSpan™ Solution component that simplifies the task of layering security onto Web services transactions. The Agent interfaces with client-side applications and automatically negotiates policy-specific security, routing, and transaction preferences with the SecureSpan™ Gateway. The Agent also manages client-side certificates issued by the Gateway, providing strong authentication, digital signing, and message-based encryption.

 

The Agent resides on the same hardware platform as the Web services client and establishes a PKI-based trust relationship with one or more Gateways. Client systems send SOAP requests to the Agent, which then negotiates the appropriate policy with the Gateway and applies any necessary protocols, headers, or transformations. Policies modified through the SecureSpan™ Manager are automatically propagated by the Gateway to the Agent. The Agent then applies any required changes to ensure that all subsequent messages conform to the updated policy.

 

The Agent transparently offloads the complexity of managing policies, keys, and certificates without adding code to the client application. This enforces the consistent application of policy while ensuring robust and flexible Web services implementation. The Agent contains two interface modes: a GUI Mode and a Troubleshooting Mode. See SecureSpan™ Agent Interface for more information.
