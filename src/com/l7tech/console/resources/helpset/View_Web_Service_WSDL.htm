View Web Service WSDL

NOTE: The SecureSpan™ Manager's optional Create WSDL Wizard guides advanced users through the WSDL creation process for defining a new Web service. See Create Web Service WSDL for more information.

The SecureSpan™ Manager allows you to view the WSDL (Web Services Description Language) document for a published Web service. WSDL is an XML-based language used to:

  • Describe the business services and transactions performed by a Web service

  • Provide electronic access instructions for users wanting to access the Web service.

Access the WSDL for a Web service in the [Web Services] tab as follows:

  1. Right-click the Web service name in the [Web Services] tab. A drop-down menu appears.

  2. Select View WSDL. The WSDL form appears.

NOTE: The WSDL in the WSDL form is read-only.

  1. View the Web service WSDL. To close the WSDL form, click the [X] in the top right corner of the form window.

NOTE: You cannot edit or replace the WSDL for a published Web service. To change the WSDL for a Web service, you must delete the Web service as outlined in Manage Published Web Services and then re-publish the Web service as outlined in Publish Web Services or Create Web Service WSDL.
