View Audit Events

In the SecureSpan™ Manager, the Gateway Audit Events window displays detailed service-specific audit messages and SecureSpan™ Gateway connectivity and communication messages. The default "info" severity level for all audit messages can be modified on a service-by-service basis using the Audit Assertion located in the Logging, Auditing, and Alerts category of the [Assertions] tab (see Configure and Edit Logging, Auditing, and Alerts Assertions).

 

The Gateway Audit Events window opens in an external SecureSpan Manager window and is divided into four sections:

  • The Audit Events window displays sortable audit messages

  • The [Details] tab displays detailed information for an auditing event selected in the Audit Events window

  • The [Associated Logs] tab displays the SecureSpan Gateway connectivity and communication messages associated with the auditing event selected in the Audit Events window

  • The area at the bottom of the Gateway Audit Events window allows you to filter information by message severity and/or enable or disable the Gateway Audit Events window's automatic refresh option.

 

The Gateway Audit Events window also allows you to download all audit events to an external file or delete audit events that are more than seven days old.

 

TIP: In the Gateway Audit Events window, click the column titles to alphabetically or numerically sort column information into ascending or descending order.

 

  1. In the Main Menu, click [Tasks] > Analyze Gateway Audit Events.

  2. The Gateway Audit Events window appears in a separate window. View, analyze, organize, download, and delete audit messages as follows:

NOTE: The Gateway Audit Events window displays the last 500 audit events.

View Event Details

NOTE: By default, all audit events are logged at the "Info" severity level. Use the Audit Assertion from the Logging, Auditing, and Alerts category of the [Assertions] tab (see Configure and Edit Logging, Auditing, and Alerts Assertions) to configure different severity levels on a service-by-service basis.

View the details of a particular event as follows:

    1. Select the target event in the Audit Events window.

    2. View the corresponding event details in the [Details] tab.

    3. View the associated SecureSpan Gateway log information for the event in the [Associated Logs] tab.

TIP: You can cut and paste the information from the [Details] and [Associated Logs] tabs into another location.

    1. Repeat to view the details of another event in the Audit Events window.

 

Filter by Message Severity

Use the filter tool located at the bottom of the Gateway Audit Events window to filter Audit Events window messages according to the message severity reported in the Severity column. Slide the filter tool to an option to immediately populate the Audit Events window with messages of that severity type. The message severity options are:

NOTE: The other message severity types in the Severity column — Fine, Finer, and Finest — only appear when the "All" option is selected in the filter tool. These messages are generally internal SecureSpan™ Gateway messages.

    • All: Slide the filter tool to the "All" option to view all event message severity types in the Audit Events window

TIP: You can download all audit events in the Audit Events window to an external location (see Download Audit Events).

    • Info: Slide the filter tool to the "Info" option to view event messages rated from "Info" to "Severe" in the Audit Events window. Information messages provide general information about an audit event

    • Warning: Slide the filter tool to the "Warning" option to view messages rated from "Warning" to "Severe" in the Audit Events window. Warning messages are alerts that may or may not require follow-up

    • Severe: Slide the filter tool to the "Severe" option to view severe or fatal problem messages in the Audit Events window. Severe messages are alerts that require immediate follow-up.

 

Set Refresh Option

NOTE: See the message in the bottom right-hand corner of the Audit Events window for specific date and time update information.

If not already checked, check the Auto-Refresh check box located at the bottom of the Gateway Audit Events window to automatically refresh audit information every five seconds.

TIP: When troubleshooting, un-check the Auto-Refresh check box to freeze Audit Events information.

 

Download Audit Events

To download all audit events to an external file:

    1. Select [File] > Download All Audit Events from the Gateway Audit Events window Main Menu.

    2. The Select File to Save form appears. Navigate to your target save location. Click [Save].

    3. A confirmation message appears. Click [OK].

NOTE: The SecureSpan™ Manager saves audit events as a pipe-delimited text file in a zip file. The file is accompanied by a digitally-signed XML file containing checksum and metadata information about the exported audit records. The XML file is signed using the SecureSpan™ Gateway's SSL certificate.

 

Delete Old Audit Events

NOTE: Audit events with a level of “Severe” cannot be deleted.

To delete audit events with a level below “Severe” that are more than seven days old:

    1. Select [File] > Delete Old Audit Events from the Gateway Audit Events window Main Menu.

    2. A confirmation message appears. Click [Delete Events].

    3. A confirmation message with a level of "Severe" appears in the Audit Events window.

 

  1. To close the Gateway Audit Events window, select [File] > Exit from the Main Menu or click the [X] in the top right corner of the window.
