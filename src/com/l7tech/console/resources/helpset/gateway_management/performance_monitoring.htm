Performance Monitoring

The Manager offers various tools to monitor the performance of the Gateway. Administrators can use these tools to plan their network resources, troubleshoot issues, and monitor the general performance of their SecureSpan™ Solution.  The monitoring tools included in the Manager are:

Logs: The Manager offers two sets of logs, the Gateway logs and the Manager logs. These logs track the messages and requests processed by each component. This data includes detailed time, severity, and message information for in-depth troubleshooting and issue resolution.

Statistics: The Manager displays detailed statistics about the attempted and completed Gateway requests. These statistics offer performance information that can be used for both network resource management and for troubleshooting.

Status bar: The status bar displays the current user name, the Gateway server being accessed, and the port number being used.
