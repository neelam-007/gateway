Logs Overview

The Manager offers two set of logs: one that displays the Gateway messages and one that displays the Manager messages. The Gateway logs are the most important set for administrative purposes. The Manager logs primarily exist as a troubleshooting tool for Layer 7 Technologies.

Gateway Log

audience - Manager administrator

purpose -

components - message #, time, severity, message, class method

To view the Gateway log

  1. Select View > Gateway log. The Log tab appears at the bottom of the user interface.

  2. Select the level of message display on the sliding scale.

  3. Select Auto-refresh to receive the most recent log data.

  4. Select Details to view a separate pane with the complete message details.

Manager Log

audience - Layer7 troubleshooting staff

purpose -

sliding scale -

To view the Manager log

  1. Select Help > Manager log. The Manager Log dialog box displays.

  2. Select the level of message display on the sliding scale.

  3. Click Dismiss to close the dialog box.
