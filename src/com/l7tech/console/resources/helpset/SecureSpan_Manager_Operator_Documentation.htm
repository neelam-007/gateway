SecureSpan™ Manager Operator Documentation

 NOTE: See SecureSpan™ Manager Documentation for Help System information and navigation instructions.

The Operator Interface of the SecureSpan™ Manager displays a limited set of view-only features and functions. This full-version SecureSpan™ Manager Help System, however, includes all features and functions available in the Administrator Interface of the Manager. Use the following table to quickly access the Help System sections relevant to the Operator Interface. See SecureSpan™ Manager Interfaces for more information about the Administrator Interface and Operator Interface.
