SecureSpan™ Manager Workflow

Several SecureSpan™ Manager components are required before you can leverage the SecureSpan™ Gateway's functionality. Since every organization will use the SecureSpan™ Manager differently, the following list includes the key tasks for configuring and using the Manager. For assistance in the configuration process, contact a Technical Support Representative as outlined in Technical Support and Contact Information.

General Workflow:

  1. The initial connection to the SecureSpan™ Gateway was established during the first launch of the SecureSpan™ Manager. The Login form with the Gateway connection(s) automatically appears every time you launch the Manager. To create or edit a Gateway connection setting, or to establish a connection to an additional Gateway, see Connect to the SecureSpan™ Gateway.

  2. Optionally configure or re-configure the Manager interface display and connection preferences as outlined in Configure SecureSpan™ Manager Preferences.

  3. Configure External Identity Providers for Web services as outlined in Add and Edit External Identity Providers.

  4. Optionally configure users and groups for the Internal Identity Provider as outlined in Add and Edit Internal Identity Provider Groups and Users.

  5. Advanced Option: Optionally create WSDL for a new Web service as outlined in Create Web Service WSDL.

  6. Publish a new Web service as outlined in Publish Web Services.

  7. Construct a valid security policy for a published Web service as outlined in Web Service Security Policies Overview.

  8. Optionally view and analyze the performance of the Gateway as outlined in SecureSpan™ Gateway Analysis Overview.
