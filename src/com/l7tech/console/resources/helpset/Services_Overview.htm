Services Overview

 NOTE: In the SecureSpan™ Manager, the term "identity" includes both users and groups, the term "user" can represent an individual human or machine, and the term "SecureSpan™ Gateway" includes all of the SecureSpan Gateway products.

The SecureSpan™ Manager differentiates between SOAP Web services and XML or non-SOAP applications. Collectively referred to as "services," each requires a different publication wizard:

  • Web services from existing WSDL (Web Services Description Language) documents are published using the Publish SOAP Web Service Wizard (see Manage SOAP Web Services)

  • Web services that require the generation of a new WSDL document are published using the Create WSDL Wizard (see Manage SOAP Web Services)

  • XML applications are published using the Publish XML Application Wizard (see Manage XML Applications).

 

Both Web services and XML applications appear in the [Services] tab upon publication. Security policies for the services are configured in the Policy Development window. For more information about policies, see Policies Overview. For information on disabling/enabling, renaming, or deleting a published service, see Manage Published Services. For information on the interface areas related to services and policies, see SecureSpan™ Manager Interfaces.
