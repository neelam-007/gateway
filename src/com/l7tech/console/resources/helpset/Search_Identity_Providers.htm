Search Identity Providers

 NOTE: In the SecureSpan™ Manager, the term "identity" includes both users and groups, and the term "user" can represent an individual human or machine.

The SecureSpan™ Manager allows you to search External Identity Provider (EIP), Federated Identity Provider (FIP), and Internal Identity Provider (IIP) users and groups. Use the Search Identity Provider form to:

  • Find and view EIP users and groups

 NOTE: External Identity Provider (EIP) users and groups are read-only in the SecureSpan™ Manager. Use the appropriate external management program to modify EIP users and groups. For information on configuring and editing EIPs in the SecureSpan™ Manager, see Add and Edit External Identity Providers.

  • Search, edit, and delete FIP users and groups. See Add and Edit Federated Identity Provider Users and Groups for more information

  • Search, edit, and delete IIP users and groups. See Add and Edit Internal Identity Provider Groups and Users for more information.

Use the Search Identity Provider Form

 NOTE: The SecureSpan™ Manager uses a definition XML file to support Tivoli (Tivoli Access Manager) External LDAP Identity Provider (EIP) directory searches. See Add and Edit External Identity Providers for more information about EIPs.

Use the Search Identity Provider form to search an External Identity Provider (EIP), Federated Identity Provider (FIP), and Internal Identity Provider (IIP) as follows:

 

  1. The Search Identity Provider form is accessed by one of two methods:

    • Click the "Search Identity Provider" link on the Home Page

 NOTE: The Home Page appears upon startup or when the Main Tool Bar's [Home] button is clicked.

    • Click [Tasks] > Search Identity Provider from the Main Menu

    • Right-click an EIP, FIP, or IIP name in the [Identity Providers] tab. A drop-down menu appears. Select Search Identity Provider.

  1. The Search Identity Provider form appears. Do the following:

    • If not already selected, select the target EIP, FIP, or IIP name from the Search drop-down list

    • Select the search target — users, groups, or both — from the Type drop-down list

    • If necessary, specify any search criteria using the Name drop-down list and field

 TIP: The Search Identity Provider form supports the following search characters in the Name field:

  • The asterisk wild card (*) retrieves terms with a common initial or subsequent spelling. It matches any string of zero or more characters when searching the External Identity Provider, Federated Identity Provider, and Internal Identity Provider

  • The question mark wild card operator (?) substitutes one character in a term. Use within a word or at the end of a word when searching the Internal Identity Provider.

    • Click [Search].

 NOTE: To stop a search that is in progress, click [Stop].

  1. Search results appear in the Search Results window. Do any of the following:

View an External Identity Provider User or Group

 NOTE: External Identity Provider (EIP) users and groups are added externally. Use the appropriate external management program to add, modify, or delete EIP users or groups. See Add and Edit External Identity Providers for more information about EIPs.

View the User Properties or Group Properties form for an External Identity Provider user or group, respectively, as follows:

    1. In the Search Results window in the Search Identity Provider form, double-click the target user or group name or select the user or group name and click [Select].

    2. The User Properties or Group Properties form appears. View user or group properties. When finished, click [OK] to close the form.

    1. Do one of the following:

      • View another EIP user or group

      • When completed, click [Close] to close the Search Identity Provider form.

View, Edit, or Delete a Federated Identity Provider or Internal Identity Provider User or Group

 NOTE: An error message will appear if you try to delete a user or group that is being used by a security policy. To delete the user or group, you must first delete the user or group from all of the service policies that it is attached to. See Configure and Edit Security Policies for more information.

To view, edit, or delete Federated Identity Provider (FIP) or Internal Identity Provider (IIP) user or groups, or to modify group membership, follow the instructions outlined in Add and Edit Federated Identity Provider Users and Groups and Add and Edit Internal Identity Provider Groups and Users, respectively.

 NOTE: At least one administrative Internal Identity Provider user must be present in the SecureSpan™ Manager. An error message will appear if you try to delete all administrative users or the last administrative user.  

 

  1. To initiate a new search, click [New Search] in the Search Identity Provider form. To close the Search Identity Provider form, click [Close].
