Search Identity Providers

 NOTE: In the SecureSpan™ Manager, the term "identity" includes both users and groups, and the term "user" can represent an individual human or machine.

The SecureSpan™ Manager allows you to search LDAP Identity Provider (LIP), Federated Identity Provider (FIP), and Internal Identity Provider (IIP) users and groups. Use the Search Identity Provider form to:

  • Find and view LIP users and groups

 NOTE: LDAP Identity Provider (LIP) users and groups are read-only in the SecureSpan™ Manager. Use the appropriate external management program to modify LIP users and groups. For information on configuring and editing LIPs in the SecureSpan Manager, see Add and Edit LDAP Identity Providers.

  • Search, edit, and delete FIP users and groups. See Add and Edit Federated Identity Provider Users and Groups for more information

  • Search, edit, and delete IIP users and groups. See Add and Edit Internal Identity Provider Groups and Users for more information.

 

Use the Search Identity Provider Form

 NOTE: The SecureSpan™ Manager uses a definition XML file to support Tivoli (Tivoli Access Manager) LDAP Identity Provider (LIP) directory searches. See Add and Edit LDAP Identity Providers for more information about LIPs.

Use the Search Identity Provider form to search an LDAP Identity Provider (LIP), Federated Identity Provider (FIP), and Internal Identity Provider (IIP) as follows:

 

  1. The Search Identity Provider form is accessed by one of two methods:

    • Click the "Search Identity Provider" link on the Home Page

    • Click [Tasks] > Search Identity Provider from the Main Menu

    • Right-click an LIP, FIP, or IIP name in the [Identity Providers] tab. A drop-down menu appears. Select Search Identity Provider.

  1. The Search Identity Provider form appears. Do the following:

    • If not already selected, select the target LIP, FIP, or IIP name from the Search drop-down list

    • Select the search target — users, groups, or both — from the Type drop-down list

    • If necessary, specify any search criteria using the Name drop-down list and field

 TIP: The Search Identity Provider form supports the following wild card characters in the Name field:

•  The asterisk truncation operator (*) retrieves terms with a common initial spelling. It substitutes a string of zero or more characters when searching the Internal Identity Provider, Federated Identity Providers, and LDAP Identity Providers

•  The question mark wild card operator (?) substitutes zero or one character in a term. Use within a word or at the end of a word when searching the Internal Identity Provider.

    • Click [Search].

  1. Search results appear in the Search Results window. Do any of the following:

View an LDAP Identity Provider User or Group

 NOTE: LDAP Identity Provider (LIP) users and groups are added externally. Use the appropriate external management program to add, modify, or delete LIP users or groups. See Add and Edit LDAP Identity Providers for more information about LIPs.

View the User Properties or Group Properties form for an LDAP Identity Provider user or group, respectively, as follows:

    1. In the Search Results window in the Search Identity Provider form, double-click the target user or group name or select the user or group name and click [Select].

    2. The User Properties or Group Properties form appears. View user or group properties. When finished, click [OK] to close the form.

    3. Do one of the following:

      • View another LIP user or group

      • When completed, click [Close] to close the Search Identity Provider form.

View, Edit, or Delete a Federated Identity Provider or Internal Identity Provider User or Group

 NOTE: An error message will appear if you try to delete a user or group that is being used by a policy. To delete the user or group, you must first delete the user or group from all of the policies that it is attached to. See Configure and Edit Policies for more information.

To view, edit, or delete Federated Identity Provider (FIP) or Internal Identity Provider (IIP) users or groups, or to modify group membership, follow the instructions outlined in Add and Edit Federated Identity Provider Users and Groups and Add and Edit Internal Identity Provider Groups and Users, respectively.

 

 NOTE: At least one administrative Internal Identity Provider user must be present in the SecureSpan™ Manager. An error message will appear if you try to delete all administrative users or the last administrative user.  

 

  1. To initiate a new search, click [New Search] in the Search Identity Provider form. To close the Search Identity Provider form, click [Close].
