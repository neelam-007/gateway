Search Identity Providers

The SecureSpan™ Manager allows you to search External Identity Provider (EIP) and Internal Identity Provider (IIP) groups and users. Use the Search Identity Provider form to:

  • Find and view EIP groups and users

NOTE: External Identity Provider (EIP) groups and users are read-only in the SecureSpan™ Manager. Use the appropriate external management program to modify EIP groups and users. For information on configuring and editing EIPs in the SecureSpan™ Manager, see Add and Edit External Identity Providers.

  • Search, edit, and delete IIP groups and users. See Add and Edit Internal Identity Provider Groups and Users for more information.

Use the Search Identity Provider Form

NOTE: The SecureSpan™ Manager uses a definition XML file to support Tivoli (Tivoli Access Manager) LDAP directory searches (External Identity Provider).

Use the Search Identity Provider form to search an External Identity Provider (EIP) or the Internal Identity Provider (IIP) as follows:

  1. The Search Identity Provider form is accessed by one of two methods:

    • Click [Tasks] > Search Identity Provider from the Main Menu

    • Right-click an EIP or IIP name in the [Identity Providers] tab. A drop-down menu appears. Select Search.

  1. The Search Identity Provider form appears. Do the following:

    • If not already selected, select the target EIP or IIP name from the Search drop-down list

    • Select the search target — groups, users, or both — from the Type drop-down list

    • If necessary, specify any search criteria using the Name drop-down list and field

NOTE: The Search Identity Provider form supports the following search characters in the Name field:

  • The asterisk truncation operator (*) retrieves terms with a common initial spelling. It substitutes a string of zero or more characters when searching the Internal Identity Provider and the External Identity Providers

  • The question mark wild card operator (?) substitutes zero or one character in a term. Use within a word or at the end of a word when searching the Internal Identity Provider.

    • Click [Search].

  1. Search results appear in the Search Results window. Do any of the following:

View an External Identity Provider Group or User

View the Group Properties form or User Properties form for an External Identity Provider group or user, respectively, as follows:

    1. In the Search Results window in the Search Identity Provider form, double-click the target group or user name or select the group or user name and click [Select].

    2. The Group Properties or User Properties form appears. View group or user properties. When completed, click [OK] to close the form.

NOTE: External Identity Provider (EIP) groups and users are read-only in the SecureSpan™ Manager. Use the appropriate management program to modify EIP groups and users.

    1. Do one of the following:

      • View another EIP group or user

      • When completed, click [Close] to close the Search Identity Provider form.

View or Edit an Internal Identity Provider Group or User

Following the instructions outlined in Add and Edit Internal Identity Provider Groups and Users to view and edit an Internal Identity Provider group or user and/or to modify group membership.

Delete an Internal Identity Provider Group or User

NOTE: An error message will appear if you try to delete a group or user that is being used by a Web service security policy. To delete the group or user, you must first delete the identity from all of the service policies that it is attached to. See Configure and Edit Security Policies for more information.

Following the instructions outlined in Add and Edit Internal Identity Provider Groups and Users to delete a group or user from the Internal Identity Provider.

NOTE: At least one administrative user must be present in the SecureSpan™ Manager. An error message will appear if you try to delete all administrative users or the last administrative user.  

  1. To initiate a new search, click [New Search] in the Search Identity Provider form. To stop a search that is in progress, click [Stop]. To close the Search Identity Provider form, click [Close].
