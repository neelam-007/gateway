Connect to the SecureSpan™ Gateway

NOTE: The initial connection to a SecureSpan™ Gateway was established during the first launch of the SecureSpan™ Manager.

The Login form allows you to:

  • Connect to a SecureSpan™ Gateway using the pre-populated URL(s) in the Gateway URL drop-down list, or

  • Establish a connection to a new Gateway.

 

NOTE: See SecureSpan™ Gateway Analysis Overview for information on analyzing the performance of one or more SecureSpan™ Gateways.

Connect to an Existing SecureSpan™ Gateway

The SecureSpan™ Manager Login form allows you to connect to an existing SecureSpan™ Gateway. The Login form:

  • Automatically appears during the launch of the Manager, and/or

  • Is accessed from within the program by one of two methods:

NOTE: You must be disconnected from a SecureSpan™ Gateway before you can connect to another Gateway. See Disconnect from a SecureSpan™ Gateway for more information.

    • Click the [Connect] button on the Main Tool Bar or,

    • Click [Session] > Connect from the Main Menu.

Use the Login form to connect to a Gateway as follows:

  1. In the Login form, enter your user name and password into the respective User Name and Password fields.

TIP: You can configure the SecureSpan™ Manager to save your user name so that you only have to enter your password in the Login form. See Configure SecureSpan™ Manager Preferences for information.

  1. Select the target Gateway from the Gateway URL drop-down list. Click [OK]. A pop-up status message appears and then automatically disappears when the Gateway connection is established.

NOTE: If a cluster of SecureSpan™ Gateways exists on your network, then there will be more than one URL in the Gateway URL drop-down list. Each URL corresponds to a different Gateway in a cluster of Gateways managed by the SecureSpan™ Manager.

Disconnect from a SecureSpan™ Gateway

NOTE: You must disconnect from a SecureSpan™ Gateway before you can connect to another Gateway or establish a new Gateway connection. The Main Tool Bar [Connect] button and the Main Menu [Session] > Connect option only become enabled when you disconnect from a Gateway.

Disconnect the current SecureSpan™ Gateway connection as follows:

  • Click the [Disconnect] button on the Main Tool Bar or,

  • Click [Session] > Disconnect from the Main Menu.

Establish a New SecureSpan™ Gateway Connection

NOTE: You must be disconnected from a SecureSpan™ Gateway before you can establish a connection to a new Gateway. See Disconnect from a SecureSpan™ Gateway for more information. The following instructions assume that the SecureSpan™ Manager is already open. You can also establish a new Gateway connection using the Login form that appears when launching the Manager.

Use the Login form to establish a new SecureSpan™ Gateway connection as follows:

  1. If necessary, disconnect the current Gateway connection as outlined in Disconnect from a SecureSpan™ Gateway.

  2. Open the Login form by one of two methods:

    • Click the [Connect] button on the Main Tool Bar or,

    • Click [Session] > Connect from the Main Menu.

NOTE: The Main Tool Bar [Connect] button and the Main Menu [Session] > Connect option only become enabled when you are disconnected from a SecureSpan™ Gateway.

  1. The Login form appears. Enter your user name and password into the respective User Name and Password fields.

TIP: You can configure the SecureSpan™ Manager to save your user name so that you only have to enter your password in the Login form. See Configure SecureSpan™ Manager Preferences for information.

  1. Enter the URL of the new Gateway into the Gateway URL field in the following format: https://machinename.domain.com:8443. Click [OK]. A pop-up status message appears and then automatically disappears when the Gateway connection is established. The SecureSpan™ Manager connects you to the new Gateway and saves the new URL in the Gateway URL drop-down list for subsequent launches.
