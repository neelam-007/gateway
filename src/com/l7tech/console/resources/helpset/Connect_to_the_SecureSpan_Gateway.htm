Connect to the SecureSpan™ Gateway

NOTE: The initial connection to a SecureSpan™ Gateway was established during the first launch of the SecureSpan™ Manager. The group assignment of your user credentials determines whether you have access to the Administrator Interface or Operator Interface (see Interfaces).

The Login form allows you to:

  • Connect to a SecureSpan™ Gateway using the pre-populated URL(s) in the Gateway URL drop-down list, or

  • Establish a connection to a new SecureSpan Gateway.

 

TIP: See Analyze SecureSpan™ Gateway Performance for information on viewing the status and log information for one or more SecureSpan™ Gateways.

Connect to an Existing SecureSpan™ Gateway

The SecureSpan™ Manager Login form allows you to connect to an existing SecureSpan™ Gateway or SecureSpan Gateway cluster. The Login form:

  • Automatically appears during the launch of the SecureSpan Manager, and/or

  • Is accessed from within the SecureSpan Manager by one of two methods:

NOTE: You must be disconnected from a SecureSpan™ Gateway (see Disconnect from a SecureSpan™ Gateway) before you can connect to another SecureSpan Gateway.

    • Click the [Connect] button on the Main Tool Bar, or

    • Click [File] > Connect from the Main Menu.

 

Use the Login form to connect to a SecureSpan Gateway as follows:

  1. In the Login form, enter your user name and password into the respective User Name and Password fields.

TIP: You can configure the SecureSpan™ Manager to save your user name so that you only have to enter your password in the Login form. See Configure Preferences for information.

  1. Select the target SecureSpan Gateway from the Gateway URL drop-down list. Click [OK]. A pop-up status message appears and then automatically disappears when the SecureSpan Gateway connection is established.

NOTE: If more than one SecureSpan™ Gateway exists, then there will be more than one URL in the Gateway URL drop-down list. Each URL corresponds to a different SecureSpan Gateway or cluster of SecureSpan Gateways managed by the SecureSpan™ Manager.

 

Disconnect from a SecureSpan™ Gateway

Disconnect the current SecureSpan™ Gateway connection as follows:

  • Click the [Disconnect] button on the Main Tool Bar or,

  • Click [File] > Disconnect from the Main Menu.

 

Establish a New SecureSpan™ Gateway Connection

NOTE: You must be disconnected from a SecureSpan™ Gateway (see Disconnect from a SecureSpan™ Gateway) before you can establish a connection to a new SecureSpan Gateway. You can also establish a new SecureSpan Gateway connection using the Login form that appears when launching the SecureSpan Manager.

Use the Login form to establish a new SecureSpan™ Gateway connection as follows:

  1. If necessary, disconnect the current SecureSpan Gateway connection as outlined in Disconnect from a SecureSpan™ Gateway.

  2. Open the Login form by one of two methods:

    • Click the [Connect] button on the Main Tool Bar or,

    • Click [File] > Connect from the Main Menu.

NOTE: The Main Tool Bar [Connect] button and the Main Menu [File] > Connect option only become enabled when you are disconnected from a SecureSpan™ Gateway.

  1. The Login form appears. Enter your user name and password into the respective User Name and Password fields.

TIP: You can configure the SecureSpan™ Manager to save your user name so that you only have to enter your password in the Login form. See Configure Preferences for information.

  1. Enter the URL of the new SecureSpan Gateway into the Gateway URL field in the following format: machinename.domain.com. Click [OK]. A pop-up status message appears and then automatically disappears when the SecureSpan Gateway connection is established. The SecureSpan™ Manager connects you to the new SecureSpan Gateway and saves the new URL in the Gateway URL drop-down list for subsequent launches.

 

NOTE: In order to enable the new SecureSpan™ Gateway, you must use the SecureSpan™ Manager's Gateway Cluster License form to install the SecureSpan Gateway license file (see Manage SecureSpan™ Gateway Licenses).
