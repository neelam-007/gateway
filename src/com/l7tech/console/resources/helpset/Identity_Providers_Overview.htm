Identity Providers Overview

Two types of identity providers are used in the SecureSpan™ Manager: multiple External Identity Providers and a single Internal Identity Provider.

External Identity Providers

Users can configure and manage one or more External Identity Providers (EIPs). An EIP is the LDAP (Lightweight Directory Access Protocol) authentication source that resides outside of the SecureSpan™ Gateway but inside an organization's firewall. The SecureSpan™ Manager allows you to configure four types of EIPs: Oracle (Oracle Internet Directory), MSAD (Microsoft Active Directory), TivoliLDAP (Tivoli Access Manager), and GenericLDAP. For information on adding and editing an EIP, see Add and Edit External Identity Providers.

Internal Identity Provider

A single Internal Identity Provider (IIP) is pre-configured as the authentication database inside the SecureSpan™ Gateway. The SecureSpan™ Manager allows you to modify the groups and users in the IIP. For information on adding groups and/or users to the IIP, see Add and Edit Internal Identity Provider Groups and Users.

Searching Identity Providers

The Search Identity Providers form permits both Internal Identity Provider and External Identity Provider searches. See Search Identity Providers for more information.
