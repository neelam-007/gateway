About Version 3.4

Version 3.4 of the SecureSpan™ Manager includes the following additions and enhancements:

  • Ability to install and view a SecureSpan™ Gateway license (see Manage SecureSpan™ Gateway Licenses)

  • New KeyInfo references in the SAML Assertion that support references with Subject Key Identifiers (see Configure and Edit Access Control Assertions)

  • HTTP POST support using form authentication in the SAML Browser Artifact assertion (see Configure and Edit Access Control Assertions)

  • A new Threat Protection category in the [Assertions] tab with two new assertions that protect against Web service and XML-specific threats. The SQL Attack Protection assertion blocks SQL attacks by looking for metacharacters inside elements and elsewhere, and the Request Size Limit assertion sets and enforces a defined limit on the size of request messages (see Configure and Edit Threat Protection Assertions).

 

Version 3.4 of the SecureSpan Manager also includes improvements to some existing features.
