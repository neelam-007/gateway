Preferences

The Preferences dialog box sets the Manager connection and GUI display preferences. This dialog box is launched after installation to establish the immediate configuration of the Manager.

Launching Preferences

After initial configuration, the Preferences dialog box can be launched via the Preferences button or by selecting Session > Preferences.

Preferences fields

The Preferences dialog box has four fields:

Server URL: Contains the server and port name for the Gateway. Ex. https://server:8443

Look and Feel: Provides four options for setting the GUI appearance. The options are: Metal, CDE/Motif, Windows, and Kunststoff.

Inactivity Timeout: The time after which the Manager will disconnect from the Gateway. Setting this value to 0 maintains the connection until the user manually disconnects.

Remember last login ID: When selected, this saves the last login ID name used in a successful login. The Login field is automatically populated when the Log In - Layer 7 Manager dialog box is launched.
