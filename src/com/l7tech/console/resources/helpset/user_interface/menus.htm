Menus

The SecureSpan Manager offers four menus to edit content and to view various Manager interfaces. The menus are: Session, Edit, View, and Help. Each menu item lists the appropriate keystroke shortcut (if available).

Session

The Session menu offers options to control the current Manager session.

Connect - Launches the Log In dialog box to connect to the SecureSpan Gateway.

Disconnect - Immediately disconnects the current session with the SecureSpan Gateway.

Preferences - Launches the Preferences dialog box. See Preferences for more information.

Exit - Disconnects the current session and closes the Manager.

Action

The Edit menu offers options to create a service and/or policy assertions.

Publish service - Launches the Publish service wizard to create a new service and its basic configuration.

Create service - Launches the Create service wizard to create a new WSDL. This tool is for advanced users only.

Create new user - Launches the New user dialog box to add the user name, password, and additional properties (if needed).

Create new group - Launches the New group dialog box to create a group and additional properties (if needed).

New provider - Launches the Edit Identity Provider dialog box to create a new provider with the name and provider type.

View

The View menu offers options to change the user interface appearance.

Refresh - Automatically refreshes the user interface to display the latest changes.

Go To > Home - Launches the home page that was displayed on first connection.

Gateway Log - Displays a tab at the bottom of the screen with Gateway server messages and message details.

Statistics - Displays a tab at the bottom of the screen with statistics about the Gateway services and requests.

Status Bar - Displays the Gateway server, port, and current user data. The status bar is located at the bottom of the user interface.

Help

The Help menu offers options to obtain further information about the Manager application.

Help Topics - Launches the Manager online help.

Manager Log - Launches the Manager Log viewer. This provides Manager message data and is used primarily as a troubleshooting tool by Layer 7 Technologies support staff.

About - Launches the About SecureSpan Manager dialog that contains version and product information. Under the Info tab, the Java Heap field refers to the total memory allocated to the Manager on installation. The memory bar refers to the total memory currently being used. The System Properties field is used primarily as a troubleshooting tool by Layer 7 Technologies support staff.
