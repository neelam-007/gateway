Screen Sections

The Manager includes seven key sections: navigation, assertions, services, policy, messages, logs and statistics, and status.

Navigation

The Navigation section of the Manager includes the menus and the button toolbar.

Assertions

The Assertions pane includes the following options:

  • identity providers

  • policy templates

  • authentication methods

  • Transport Layer Security (TLS)

  • XML security

  • XML transform & validation

  • Message routing

Services

The Services pane displays the list of published Web services in alphabetical order. These services are enabled by default, but can easily be disabled to prevent access to the service. Disabled services display with a red X through the service icon. The service details are listed under each individual service including the messages, bindings, port types, and services.

Policy

The Policy pane lists the service's policy assertions. This pane is used primarily to edit and manage security policies. The policy buttons support the key policy tasks which are: Save policy, Save policy template, Validate policy, and Toggle view.

Messages

The Messages pane displays policy validation and save messages.

Logs and Statistics

The Logs and Statistics tabs display message and request statistics for performance monitoring and troubleshooting assistance. This pane is visible when the Gateway Log and Statistics options are selected under the View menu.

Status Bar

The Status Bar displays the Gateway server, port, and current user information. This pane is visible when the Status Bar option is selected under the View menu.
