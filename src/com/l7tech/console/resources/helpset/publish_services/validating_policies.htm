Validating Policies

Once the assertions have been added to a policy, the validation process is critical to ensure that the policy works. Validating and fixing policies is a fast and simple process in the Manager. Validation takes only a few seconds and displays any errors in the assertion ordering and/or combining.

 

To validate a policy:

  1. Add the appropriate assertions to the policy.

  2. Once complete, click the Validate policy button.

  3. Check the Messages section for any errors. If errors appear, go to step 4. If the policy is "ok," the validation is complete.

  4. When errors appear in the validation process, verify that the assertions are correctly ordered and that the combinations are acceptable. For more information, see the Creating Policies topic.
