Audience

The SecureSpan™ Manager online help is intended for system administrators of the SecureSpan™ Manager. This documentation assumes a general knowledge of your client operating system, internetworking terminology,  http security technologies, and Web services. The online help does not include installation and configuration procedures, product specifications, or external product information.  

WSDL Creation Wizard

Administrators who use the WSDL Web services creation wizard require additional knowledge. This wizard supports complex functionality such as service definition, establishing operation messages, and defining SOAPAction operation attributes. As a result, the wizard is primarily designed for architects. Administrators and architects using this wizard must have a deep understanding of the WSDL standard, SOAP, XML, and the SOAP protocols.
