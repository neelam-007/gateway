Manage Published Web Services

NOTE: You cannot edit or replace the WSDL for a published Web service. To change the WSDL for a Web service, you must delete the Web service as outlined in Delete a Web Service and then re-publish the Web service as outlined in Publish Web Services or Create Web Service WSDL.

In the SecureSpan™ Manager, each Web service in the [Web Services] tab contains a drop-down menu of management options that allow you to disable, enable, rename, and delete the Web service. For more information about the [Web Services] tab, see SecureSpan™ Manager Interface.

Disable/Enable a Web Service

By default, all Web services are enabled after publication. The SecureSpan™ Manager allows you to enable and disable a Web service. A disabled Web service cannot be accessed.

Disable a Web Service

To disable a Web service:

  1. Right-click the Web service name in the [Web Services] tab. A drop-down menu appears.

  2. Select Disable. A red "X" appears through the disabled Web service icon in the [Web Services] tab.

Enable a Web Service

A disabled Web service contains a red "X" through its Web service icon. To enable a disabled Web service:

  1. Right-click the Web service name in the [Web Services] tab. A drop-down menu appears.

  2. Select Enable. The red "X" through the Web service icon in the [Web Services] tab disappears. The Web service is now active and accessible.

Rename a Web Service

The SecureSpan™ Manager displays published Web services alphabetically in the [Web Services] tab. Rename a Web service as follows:

  1. Right-click the Web service name in the [Web Services] tab. A drop-down menu appears.

  2. Select Rename. The Rename Web Service form appears.

  3. Enter a new name for the Web service into the Name field. Click [OK]. The new Web service name immediately appears in the [Web Services] tab.

Delete a Web Service

NOTE: You cannot undo a deleted Web service. As a safe alternative to removal, disable a Web service as outlined in Disable/Enable a Web Service instead of deleting it.

The SecureSpan™ Manager allows you to delete a published Web service. Deleting a Web service deletes the Web service properties, settings, and service policy from the SecureSpan™ Manager and the SecureSpan™ Gateway.

  1. Right-click the Web service name in the [Web Services] tab. A drop-down menu appears.

  2. Select Delete. A pop-up confirmation message appears. Click [Yes]. The Web service is immediately removed from the [Web Services] tab.

 

NOTE: See View Web Service WSDL for information on viewing the WSDL document for a published Web service. See Web Service Security Policies Overview for information on viewing, configuring, and editing a Web service security policy.
