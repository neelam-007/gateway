Welcome to the SecureSpan™ Manager

Welcome to Layer 7 Technologies' SecureSpan™ Manager Help System. This Help System provides extensive information and user instructions for the SecureSpan Manager, including:

 

  • A list of features and functions available in the Operator Interface and a quick-access list to Operator-specific documentation (see Operator Documentation)

  • Instructions for connecting to one or more SecureSpan™ Gateways (see Connect to the SecureSpan™ Gateway)

  • Instructions for connecting to LDAP Identity Providers and for configuring Internal Identity Provider users and groups (see Manage Identity Providers)

  • Instructions for publishing and managing SOAP Web services and XML (non-SOAP) applications (see Manage Services)

  • Policy assertion and policy configuration information and instructions (see Manage Policies)

  • Instructions for monitoring and auditing the SecureSpan Gateway from the SecureSpan Manager (see Analyze SecureSpan™ Gateway Performance)

  • Instructions for configuring identity bridging in the SecureSpan Manager and SecureSpan™ Bridge (see Identity Bridging)

  • Technical support contact information (see Technical Support and Contact Information).

 

For information about this Help System, see Help System Navigation.
