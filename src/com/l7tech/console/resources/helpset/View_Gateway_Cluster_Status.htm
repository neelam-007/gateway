View Gateway Cluster Status

 NOTE: The Status Bar under the [Services] tab contains a user, SecureSpan™ Gateway, and node information message when [View] > Status Bar is selected from the Main Menu (check box checked). See SecureSpan™ Manager Interfaces for more information.

In the SecureSpan™ Manager, the external Gateway Cluster Status window displays the status of the SecureSpan™ Gateway cluster node(s) and service statistics. The Gateway Cluster Status window contains two windows:

  • The Gateway Status window displays node information and CPU and server statistics by SecureSpan™ Gateway node

  • The Service Statistics window displays service activity statistics.

 

 NOTE: Data in the Gateway Cluster Status window is updated by the SecureSpan™ Gateway every five seconds. See the message in the bottom right-hand corner of the window for specific date and time update information. In the Gateway Cluster Status window, click a column title to alphabetically or numerically sort column information into ascending or descending order.

 

  1. In the Main Menu, click [Tasks] > View Cluster Status.

  2. The Gateway Cluster Status window appears in a separate window. Analyze SecureSpan™ Gateway node activity and/or service statistics as follows:

Gateway Status Window

 NOTE: The Load Sharing %, Request Routed %, and Load Avg columns are based on data from the last 60-second time period.  

The Gateway Status window displays information by Gateway cluster node. You can also rename or delete a node that appears in the Gateway Status window.

Column Name

Description

Gateway Node

Name of the cluster node assigned during configuration (SSG1, SSG2, etc). Displays three status icons:

: Node is active.

: Node is inactive.

: Node status is undetermined. The Manager is in the process of assessing the status of the node, and will change the icon to active or inactive once the status is determined.

 

Optional: Rename a node or delete an inactive node as outlined below.

Load Sharing %

Indicates the percentage of total cluster traffic being handled by the node. A value of "0" indicates no activity.

Request Routed %

Indicates the percentage of current routing activity being handled by the node. A value of "0" indicates no activity.

Load Avg

The average number of work processes completed over the last 60-second period. If the number reaches or exceeds 4.0, then the SecureSpan™ Gateway is under a heavy load and overall server performance will be slow.

Uptime

Server start time to the current time. Use this information to analyze the number of requests processed by the node per time period.

IP Address

The IP address for the direct Gateway to Gateway connection.

Rename a Node

 NOTE: Renaming a node only changes how the name is displayed in the SecureSpan™ Manager. The host name of the node will not change in the SecureSpan™ Gateway.

Rename the node that appears in the Gateway Node column as follows:

 

    1. Right-click any column field that belongs to the node. A drop-down menu appears.

    2. Select [Rename Node]. The Rename Gateway Node form appears.

    3. Enter a new name into the New Node Name field. Click [OK].

    4. The name of the node is immediately changed in the Gateway Node column and throughout the SecureSpan™ Manager.

Delete a Node

 NOTE: A deleted node is not permanently removed from the database or the SecureSpan™ Gateway cluster. When the deleted node is reactivated, stop and restart the applicable Gateway to view the node in the Gateway Status window.

Delete an inactive () node from the current session of the Gateway Status window as follows:

 

    1. Right-click any column field that belongs to the node. A drop-down menu appears.

 NOTE: An error message will appear if you try to delete an active node () from the Gateway Status window.

    1. Select [Delete Node]. A confirmation message appears. Click [Yes]. The node is immediately removed from the Gateway Status window.

Service Statistics Window

The Service Statistics window displays information by service:

Column Name

Description

Service Name

Name of the service assigned during configuration.

Requests Attempted

Raw number of request messages received by the service. See Note below.

Authorized

Raw number of request messages that have been authorized by the SecureSpan™ Gateway.

Routed

Raw number of request messages that have been successfully routed.

Routed (last min)

Raw number of request messages that have been successfully routed in the last minute of uptime.

 

 NOTE: The Requests Attempted column may be inflated if one or more services are using the HTTP Digest authentication assertion. In the HTTP Digest authentication method, a typical request includes two request messages: The first is a challenge request and the second is the actual client request message. For more information on the HTTP Digest assertion, see Configure and Edit Authentication Assertions. Other authentication methods use a single message per request.

 

  1. To close the Gateway Cluster Status window, click [File] > Exit or click the [X] in the top right corner of the window.

 

 NOTE: Re-starting a SecureSpan™ Gateway resets its statistics to zero.
