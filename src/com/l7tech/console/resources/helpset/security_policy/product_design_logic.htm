Product Design Logic

Layer 7 Technologies designed its SecureSpan™ Solution as an engine for building and implementing policy. The Manager provides the interface to administrator to build these policies. Policy consists of a number of assertions, grouped together with conjunctions, that fully describe a processing language for Web services transactions. An administrator can initially construct a policy using the wizard interfaces, which greatly simplifies the creation of policy for simple Web services. Any policy can then be viewed and edited as a tree of assertions.
