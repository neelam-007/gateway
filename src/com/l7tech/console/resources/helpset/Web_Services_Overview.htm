Web Services Overview

The SecureSpan™ Manager is primarily used for building and enabling Web service security policies. Before a security policy can be defined, the Web service must be published using the Manager's Publish Web Service Wizard (see Publish Web Services for more information).

 

A Web service is described using a WSDL (Web Services Description Language) document that is typically created elsewhere and imported into the SecureSpan™ Gateway during the publication process. Advanced users can also use the Manager's Create WSDL Wizard to generate WSDL (Web Services Description Language) for a new Web service (see Advanced Option: About the Create WSDL Wizard below).

 

For more information about security policies, see Web Service Security Policies Overview. For information on disabling/enabling, renaming, or deleting a published Web service, see Manage Published Web Services. For information on the interface areas related to Web services and security policies, see SecureSpan™ Manager Interface.

Advanced Option: About the Create WSDL Wizard

Web services are described by WSDL documents. The SecureSpan™ Manager's Create WSDL Wizard allows users to create the WSDL for a new Web service in several managed and logical steps. See Create Web Service WSDL for more information.
