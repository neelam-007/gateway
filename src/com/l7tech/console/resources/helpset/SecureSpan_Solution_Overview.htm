SecureSpan™ Solution Overview

 TIP: See www.layer7tech.com for more information about the SecureSpan™ Solution.

The SecureSpan™ Solution enables the simple creation and management of Web service and XML application security policies. Three architectural components combine to form a powerful security solution: the SecureSpan™ Gateway, the SecureSpan™ Manager, and the SecureSpan™ Bridge.

 

 NOTE: The SecureSpan™ Solution supports the XML 1.0, SOAP 1.1, SSL 2.0/3.0, HTTP 1.1, WS-I BSP 1.0, WS-Trust, WS-SC, WS-Metadata, WS-Security 1.0, PKSC #10 key store format, and X.509 v3 certificate standards. Additional supported standards and tool kits are referenced with the feature or function that they apply to.

SecureSpan™ Gateway

The SecureSpan™ Gateway is the centerpiece of the SecureSpan™ Solution’s threat protection capabilities. The Gateway sits at the boundary of a security domain, gating inbound access and regulating outbound communication. This powerful appliance performs various XML application and Web services firewalling and threat protection functions, including access management, privacy enforcement, data validation, routing, transformation, and auditing. For information about monitoring and auditing the Gateway from the SecureSpan™ Manager, see SecureSpan™ Gateway Analysis Overview.

SecureSpan™ Manager

 NOTE: For program and release information, click [Help] > About from the Main Menu to open the About the SecureSpan™ Manager form. In the form, the [About] tab contains program and release information, and the [Info] tab contains dynamic memory and system information.

The SecureSpan™ Manager is a comprehensive policy management application for defining, verifying, and managing security policies for protected Web services and XML applications. The rich palette of policy assertions provides fine-grained control of security parameters on a service-by-service basis, enabling legitimate access and screening out potential threats. The user-friendly Manager GUI (general user interface) transforms service policy configuration and management from a complex programming challenge into a simple administrative task.

 

The Manager offers two modes: 1) a GUI for all configuration and management tasks (see SecureSpan™ Manager Interfaces) and 2) a Troubleshooting Mode for assessing issues with the SecureSpan™ Gateway (see SecureSpan™ Manager Troubleshooting Mode). The Manager GUI contains two role-based functionality sets: a full-featured and privileged set for Administrators and a partial view-only set for Operators. For more information about the Manager interfaces, see SecureSpan™ Manager Interfaces. For general information about the Manager, see SecureSpan™ Manager Overview.

SecureSpan™ Bridge

 NOTE: The requestor-side SecureSpan™ Bridge is required to configure identity bridging. See Identity Bridging Overview for more information.

The SecureSpan™ Bridge coordinates security policies defined in the SecureSpan™ Manager with client applications residing in different security domains. The Bridge automatically establishes a PKI-based trust relationship with one or more SecureSpan™ Gateways. The complexity associated with the management of keys, certificates, and policies is off-loaded entirely by the Bridge so that no client application coding is required. For more information about the Bridge, see SecureSpan™ Bridge Overview.
