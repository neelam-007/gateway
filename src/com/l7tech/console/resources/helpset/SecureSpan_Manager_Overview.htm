SecureSpan™ Manager Overview

 NOTE: The term "identity" includes both users and groups, the term "user" can represent an individual human or machine, the term “service” includes both Web services and XML applications, and the term "SecureSpan™ Gateway" includes all of the SecureSpan Gateway products.

The SecureSpan™ Manager is the user interface for the SecureSpan™ Gateway. Located on the internal LAN (local area network), the SecureSpan Manager communicates with the SecureSpan Gateway over default ports. Use the SecureSpan Manager’s standards-based functionality to construct Web service and XML application policies, manage policy users, configure identity bridging, and configure, audit, and monitor the SecureSpan Gateway.

 

The SecureSpan Manager includes two different interface functionality sets — a full-featured and privileged set for Administrators and a partial read-only feature set for Operators (see Interfaces). The Microsoft Windows version of the SecureSpan Manager offers two modes: 1. A GUI for all configuration and management tasks (see Interfaces) and 2. A Troubleshooting Mode used by Network Administrators to gather additional information about system errors (see Troubleshooting Mode). A separate Troubleshooting Mode for the Linux version of the SecureSpan Manager is not available.  

 

For information on the general SecureSpan Manager workflow, see General Workflow.
