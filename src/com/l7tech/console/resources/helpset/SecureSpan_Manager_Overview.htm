SecureSpan™ Manager Overview

 NOTE: In the SecureSpan™ Manager, the term "identity" includes both users and groups, and the term "user" can represent an individual human or machine.

The SecureSpan™ Manager is the user interface for the SecureSpan™ Gateway. Located on the internal LAN (local area network), the Manager communicates with the Gateway over default ports. Use the Manager’s WS-Security standards-based functionality to construct Web service and XML application security policies, manage policy users, configure identity bridging, and configure, audit, and monitor the Gateway.

 

The Manager includes two different interface functionality sets — a full-featured and privileged set for Administrators and a partial read-only feature set for Operators. For information on the Manager interfaces, see SecureSpan™ Manager Interfaces. For information on the general user workflow, see SecureSpan™ Manager Workflow. For information on the Gateway, see SecureSpan™ Solution Overview.

 

 NOTE: For program and release information, click [Help] > About from the Main Menu to open the About the SecureSpan™ Manager form. In the form, the [About] tab contains program and release information, and the [Info] tab contains dynamic memory and system information.
