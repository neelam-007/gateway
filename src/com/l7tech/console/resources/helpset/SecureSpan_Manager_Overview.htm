SecureSpan™ Manager Overview

NOTE: For program and release information, click [Help] > About from the Main Menu to open the About the SecureSpan™ Manager form. In the form, the [About] tab contains program and release information, and the [Info] tab contains dynamic memory and system information.

The SecureSpan™ Manager is a Java-based application that is the user interface for the SecureSpan™ Gateway. Located on the internal LAN, the Manager communicates with the Gateway over default ports. Use the Manager’s WS-Security standards-based functionality to construct Web service security policies, manage policy users, and configure and monitor the Gateway.

 

For information on the Manager interface, see SecureSpan™ Manager Interface. For information on the general user workflow, see SecureSpan™ Manager Workflow. For information on the Gateway, see SecureSpan™ Solution Overview.
