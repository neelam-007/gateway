General Workflow

 NOTE: The following workflow applies to the Administrator Interface. See Operator Documentation for a list of the features available in the Operator Interface. For more information about the SecureSpan™ Manager interfaces, see Interfaces.

Several SecureSpan™ Manager components are required before you can leverage the SecureSpan™ Gateway's functionality. Since every organization will use the SecureSpan Manager differently, the following list includes the key tasks for configuring and using the SecureSpan Manager. For assistance in the configuration process, contact a Technical Support Representative as outlined in Technical Support.

 

  1. The initial connection to the SecureSpan™ Gateway was established during the first launch of the SecureSpan™ Manager. The Login form with the SecureSpan Gateway connection(s) automatically appears every time you launch the SecureSpan Manager. To create or edit a SecureSpan Gateway connection setting, or to establish a connection to an additional SecureSpan Gateway, see Connect to the SecureSpan™ Gateway.

  2. Optionally configure the SecureSpan Manager interface display and connection preferences as outlined in Configure Preferences.

  3. Each SecureSpan Gateway Internal Identity Provider (IIP) is pre-configured with two default access groups (a "Gateway Administrators" group and a "Gateway Operators" group) and a single default administrative user ("admin"). Optionally configure additional users and groups for the IIP as outlined in Add and Edit Internal Identity Provider Users and Groups.

  4. Configure LDAP Identity Providers as outlined in Add and Edit LDAP Identity Providers.

  5. Publish a new SOAP Web service as outlined in Manage SOAP Web Services and/or publish an XML application (or a non-SOAP service) as outlined in Manage XML Applications.

  6. Construct a valid policy for a published service as outlined in Manage Policies.

  7. Optionally view, audit, and analyze the performance of the SecureSpan Gateway as outlined in Analyze SecureSpan™ Gateway Performance.

 

 NOTE: For information on the steps required to configure identity bridging in the SecureSpan™ Manager and SecureSpan™ Bridge, see Identity Bridging.
