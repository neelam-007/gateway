Manage Cluster Properties

The Manage Cluster-Wide Properties form allows you to add, edit, and view the properties of a SecureSpan™ Gateway cluster. The form displays a map of string property names to string property values, such as SecureSpan Gateway licenses (see Manage SecureSpan Gateway Licenses), that are stored in the database and available throughout the cluster. Edit a Property

Edit a SecureSpan™ Gateway cluster property:

  1. In the SecureSpan Manager, select Tasks > Manage Cluster-Wide Properties from the Main Menu. The Manage Cluster-Wide Properties form appears.

  2. Click [Edit]. The Edit Cluster Properties form appears.

  3. Modify the field values as required. Click [OK].

  4. The modified property is updated in the Manage Cluster-Wide Properties form. Click [Close] to close the Manage Cluster-Wide Properties form.

 

Delete a Property

Edit a SecureSpan™ Gateway cluster property:

  1. In the SecureSpan Manager, select Tasks > Manage Cluster-Wide Properties from the Main Menu. The Manage Cluster-Wide Properties form appears.

  3. Modify the field values as required. Click [OK].
