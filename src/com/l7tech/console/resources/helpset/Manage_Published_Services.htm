Manage Published Services

 NOTE: For information on changing the WSDL for a published service, see Publish SOAP Web Services. For information on changing the Gateway URL for a published XML application, see Publish XML Applications.

In the SecureSpan™ Manager, the [Services] tab contains published Web services and XML applications. Collectively referred to as "services," each service type is differentiated by the icon to the left of the service name:

  • XML applications are identified by an .

Each service in the [Services] tab contains a drop-down menu of management options that allow you to disable, enable, rename, and delete the service.

Disable/Enable a Service

By default, all services are enabled after publication. The SecureSpan™ Manager allows you to enable and disable a service. A disabled service is inactive and inaccessible.

Disable a Service

To disable a service:

 

  1. Right-click the service name in the [Services] tab. A drop-down menu appears.

  2. Select Disable. A red "X" appears through the disabled service icon in the [Services] tab.

Enable a Web Service

A disabled service contains a red "X" through its service icon. To enable a disabled service:

 

  1. Right-click the service name in the [Services] tab. A drop-down menu appears.

  2. Select Enable. The red "X" through the service icon in the [Services] tab disappears. The service is now active and accessible.

Rename a Service

The SecureSpan™ Manager displays published services alphabetically in the [Services] tab. Rename a service as follows:

 

  1. Right-click the service name in the [Services] tab. A drop-down menu appears.

  2. Select Rename. The Rename Service form appears.

  3. Enter a new name for the service into the Name field. Click [OK]. The new service name immediately appears in the [Services] tab.

Delete a Service

 NOTE: You cannot undo a deleted service. As a safe alternative to removal, disable a service as outlined in Disable/Enable a Service instead of deleting it.

The SecureSpan™ Manager allows you to delete a published service. Deleting a service deletes the service properties, settings, and service policy from the Manager and the SecureSpan™ Gateway.

 

  1. Right-click the service name in the [Services] tab. A drop-down menu appears.

  2. Select Delete. A pop-up confirmation message appears. Click [Yes]. The service is immediately removed from the [Services] tab.
