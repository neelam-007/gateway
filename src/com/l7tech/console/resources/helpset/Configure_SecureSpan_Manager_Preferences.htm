Configure SecureSpan™ Manager Preferences

NOTE: Preferences can be configured at any time when connected or disconnected from a SecureSpan™ Gateway.

The SecureSpan™ Manager allows you to configure the following program preferences:

  • Interface appearance

  • Timeout settings

  • Saved Login form user name.

Configure Interface Appearance

The SecureSpan™ Manager allows you to choose from eight interface display options: Metal, CDE/Motif, Windows, Kunststoff, JGoodies Window, JGoodies Plastic, JGoodies Plastic 3D, and JGoodies Plastic XP. To change the appearance of the Manager interface:

  1. Launch the Preferences form by one of two methods:

    • Select [Session] > Preferences from the Main Menu, or

    • Click the [Preferences] button on the Main Tool Bar.

  1. The Preferences form appears. Select a display option from Interface Display drop-down list, then do one of the following:

    • To configure other program preferences, proceed to the sections outlined below, or

    • If configuration is complete, click [OK]. Interface changes are completed the next time you launch the Manager.

Configure Timeout Setting

For security purposes, the SecureSpan™ Manager automatically disconnects users when they have been idle on the system for a particular period of time. This period of time, called the timeout limit, is not an absolute time: it is the period of inactivity (from 1 to 60 minutes) allowed by the system before it will require a user to log back in. Set inactivity timeout limits as follows:

  1. Launch the Preferences form by one of two methods:

    • Select [Session] > Preferences from the Main Menu, or

    • Click the [Preferences] button on the Main Tool Bar.

  1. The Preferences form appears. Enter a numerical value into the Inactivity Timeout field. The value must be a whole number between "0" and "60" corresponding to the number of minutes that the Manager will wait to disconnect an inactive user. When completed, do one of the following:

TIP: Set the value in the Inactivity Timeout field to "0" to maintain an indefinite connection until you manually disconnect.

    • To configure other program preferences, proceed to the sections outlined above and below, or

    • If configuration is complete, click [OK]. Timeout setting changes are implemented immediately.

Configure Saved User Name

The SecureSpan™ Manager allows you to save the last Login form user name used to connect to a SecureSpan™ Gateway as follows:

  1. Launch the Preferences form by one of two methods:

    • Select [Session] > Preferences from the Main Menu, or

    • Click the [Preferences] button on the Main Tool Bar.

  1. The Preferences form appears. Check the Remember Last Login ID check box and then do one of the following:

    • To configure other program preferences, proceed to the sections outlined above, or

    • If configuration is complete, click [OK]. Changes are implemented the next time you connect to a Gateway.

NOTE: Un-check the Remember Last Login ID check box to deactivate the configuration.
