Manage SecureSpan™ Gateway Licenses

NOTE: Only the Administrator Interface of the SecureSpan™ Manager permits access to the Gateway Cluster License form.  

Layer 7 Technologies provides clients with a signed XML license file that permits access to a specific SecureSpan™ Gateway for a pre-determined license validity period. The license file must be installed using the SecureSpan™ Manager's Gateway Cluster License form during the first connection to a SecureSpan Gateway or SecureSpan Gateway cluster.

 

The SecureSpan Gateway or SecureSpan Gateway cluster runs in license-enabled mode as long as a valid license is installed. In license-enabled mode, the SecureSpan Gateway processes incoming service messages, provides ancillary services such as WSDL and policy discovery, and provides full administrative services to any connected SecureSpan Manager. If a valid license does not exist or is expired, then most SecureSpan Manager features will be disabled until a valid license is installed. A license can be viewed at any time from the SecureSpan Manager [Help] menu (see View License).

Install a License

NOTE: The following instructions apply whether installing a new license or replacing an existing license in the SecureSpan™ Gateway.

A SecureSpan™ Gateway license is installed through the SecureSpan™ Manager when:

  • Connecting to a new SecureSpan Gateway or SecureSpan Gateway cluster

NOTE: From this point forward, the term "SecureSpan™ Gateway" includes both individual and clustered SecureSpan Gateway implementations.

  • Replacing an expired license

  • Replacing an active license with another that contains different permissions.

 

Layer 7 Technologies provides a signed XML license file for each SecureSpan Gateway in an implementation. The license includes the SecureSpan Gateway name, a unique license key, organization information, and a set validity period. The license must be installed to enable the SecureSpan Manager's functionality when connecting to a new SecureSpan Gateway. If an existing license expires, then the SecureSpan Manager's functionality will be disabled until a valid license is installed.

 

WARNING: Do not modify the license XML. If the license file provided by Layer 7 Technologies is changed, then the SecureSpan™ Gateway will not be able to verify the signature and install the license.

 

  1. Connect to the existing or new SecureSpan Gateway associated with the license as outlined in Connect to the SecureSpan Gateway. Enter Administrator credentials into the Login form. Click [OK].

  2. Select [Help] > Manage Gateway License from the Main Menu. The Gateway Cluster License form appears.

  3. Click [Install New License]. The Select License File to Install form appears.

  4. Navigate to the location of the XML license file. Select the file. Click [Open]. One of the following will occur:

Replacing an Existing License

If replacing an existing license, then the Destroy Existing License form appears:

    • Click [Destroy Existing License] to proceed with the replacement

    • The License Status field in the Gateway Cluster License form changes to "Valid." Proceed to Step #5 below.

Installing a License for a New SecureSpan™ Gateway

If installing a license for a new SecureSpan Gateway, then the License Status field in the Gateway Cluster License form changes from "None installed" to "Valid." Proceed to Step #5 below.

  1. Click [OK]. The SecureSpan Gateway is now in license-enabled mode.

 

NOTE: It takes approximately 10 minutes for the system to update the licenses for all of the SecureSpan™ Gateways in a cluster.

 

View License

NOTE: You must be logged into a SecureSpan™ Gateway to view its license from the SecureSpan™ Manager.

In the SecureSpan™ Manager, select [Help] > Manage Gateway License from the Main Menu. The Gateway Cluster License form appears. View SecureSpan™ Gateway license details. Click [Close] to close the Gateway Cluster License form.
