Main Menu:

NOTE: Many Main Menu items can be accessed through keyboard shortcuts where provided. Press [Alt] on your keyboard to enable the shortcuts. The initiating letter for a shortcut is underlined in the Main Menu and in each respective drop-down menu. For example, to open the Preferences form (see Configure SecureSpan™ Manager Preferences) using shortcuts, click [Alt + S] (to open the [Session] menu) and then [Alt + P] (to open the Preferences form) on your keyboard.

Contains drop-down menus that include all of the program features, functions, and information.

Main Tool Bar:

Contains shortcut buttons to commonly-used program features.

Home Page:

Contains the Home Page upon startup or when the Main Tool Bar's [Home] button is clicked. The Home Page contains shortcut links to commonly used wizards and forms.

Policy Development Window:

  • Contains the Policy Development window when viewing a Web service from the [Web Services] tab. The Policy Development window is the repository for the assertions selected from the [Assertions] tab and configured into a security policy for the selected Web service. The Policy Development window includes the Policy Tool Bar that contains buttons for saving and validating a policy. See Web Service Security Policies Overview for more information

  • Contains the Policy Validation Messages window when [View] > Policy Messages is selected from the Main Menu (check box checked) and when viewing a Web service security policy in the Policy Development window. The Policy Validation Messages window contains configuration, warning, and error feedback messages for the security policy. To remove the Policy Validation Messages window from the Policy Development window, select [View] > Policy Messages from the Main Menu (check box unchecked). See Configure and Edit Security Policies for more information

  • Contains the Identity window when [Identity View] is selected from the Policy Tool Bar when viewing a Web service security policy in the Policy Development window. To toggle back to the Policy Development window, click [Policy View] on the Policy Tool Bar.

[Identity Providers] Tab:

Contains the Internal Identity Provider and a list of configured External Identity Providers. Right-click the Internal Identity Provider to search, view, and add groups and/or users. Right-click an External Identity Provider to search, view, and edit its properties. For more information, see Identity Providers Overview.

[Web Services] Tab:

Contains an alphabetical list of published Web services, each with an expandable list of the Web service messages, port types, bindings, and services. The Web services are enabled by default when published using the Publish Web Service Wizard, but can be disabled by right-clicking the Web service name and selecting [Disable] from the drop-down menu. Disabled Web services contain a red "X" through the Web service icon. For more information on Web services, see Web Services Overview. For more information on managing the published Web services in the [Web Services] tab, see Manage Published Web Services.

[Assertions] Tab:

  • Contains a categorized list of the policy assertions used to construct a security policy in the Policy Development window for a Web service selected in the [Web Services] tab. Each assertion category contains an expandable list of assertion options. See Security Policy Assertions Overview for more information

  • Includes the Assertion Tool Bar that contains shortcut buttons used to move and organize the assertions in the Policy Development window. The action commands are also available by right-clicking an assertion in the Policy Development window.

Status Bar:

Contains a user, SecureSpan™ Gateway, and port information message when [View] > Status Bar is selected from the Main Menu (check box checked). To remove the Status Bar, click [View] > Status Bar from the Main Menu (check box unchecked).

Wizards

The SecureSpan™ Manager employs user-friendly wizards to help users configure complex tasks in several managed and logical steps. Each wizard interface include three areas:

Steps List:

Presents a numbered list of the steps required by the task. Each step in the Steps list corresponds to a screen in the wizard.

Tips Box:

Some wizards contain a text box that describes the principles of or the form elements within the wizard screen.

Form:

Contains the form elements or actions required by the configuration process.
