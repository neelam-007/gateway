View Gateway Log

NOTE: The Status Bar under the [Web Services] tab contains a user, SecureSpan™ Gateway, and node information message when [View] > Status Bar is selected from the Main Menu (check box checked). See SecureSpan™ Manager Interface for more information.

In the SecureSpan™ Manager, the Gateway Log displays connectivity and communication messages for one or more SecureSpan™ Gateways. The Gateway Log opens in an external Manager window and is divided into three sections:

  • The Log Browser window displays sortable Gateway messages

  • The [Details] tab displays detailed information for a message selected in the Log Browser window

  • The area under the [Details] tab allows you to filter information by message severity and/or enable or disable the Gateway Log's automatic refresh option.

NOTE: In the Gateway Log, click the column titles to alphabetically or numerically sort column information into ascending or descending order.

  1. In the Main Menu, click [Tasks] > Analyze Gateway Log.

  2. The Gateway Log appears in a separate window. View, analyze, and organize SecureSpan™ Gateway performance messages as follows:

NOTE: The Log Browser window displays the last 500 log messages.

Column Name

Description

Node

Displays the SecureSpan™ Gateway node that the message applies to.

Time

Displays the time that the event took place in the Gateway.

Severity

Displays one of six message severity ratings assigned by the Gateway. Four of the six message types can be filtered. See below for more information.

Message

The actual Gateway event message.

View Message Details

View the details of a particular message as follows:

    1. Select the target message in the Log Browser window.

    2. View the corresponding message details in the [Details] tab.

TIP: You can cut and paste the information from the [Details] tab into another location.

    1. Repeat to view the details of another message in the Log Browser window.

Filter by Message Severity

NOTE: The other message severity types in the Severity column — Fine, Finer, and Finest — only appear when the "All" option is selected in the filter tool. These messages are generally internal SecureSpan™ Gateway messages.

Use the filter tool located under the [Details] tab to filter Log Browser messages according to the message severity reported in the Severity column. Slide the filter tool to an option to immediately populate the Log Browser window with messages of that severity type. The message severity options are:

    • All: Slide the filter tool to the "All" option to view all message severity types in the Log Browser window

    • Info: Slide the filter tool to the "Info" option to view messages rated from "Info" to "Severe" in the Log Browser window. Information messages provide general information about a SecureSpan™ Gateway event. For example, "Revoking cert and password for user"

    • Warning: Slide the filter tool to the "Warning" option to view messages rated from "Warning" to "Severe" in the Log Browser window. Warning messages are alerts that may or may not require follow-up. For example, "No authorization header found"

    • Severe: Slide the filter tool to the "Severe" option to view severe or fatal problem messages in the Log Browser window. Severe messages are alerts that require immediate follow-up. For example, "Attempt to authenticate using unsupported method."

Set Refresh Option

NOTE: See the message in the bottom right-hand corner of the Gateway Log window for specific date and time update information.

If not already checked, check the Auto-Refresh check box under the [Details] tab to automatically refresh SecureSpan™ Gateway information every five seconds.

 

TIP: When troubleshooting, un-check the Auto-Refresh check box to freeze Gateway Log information.

  1. To close the Gateway Log, select [File] > Exit or click the [X] in the top right corner of the log window.
