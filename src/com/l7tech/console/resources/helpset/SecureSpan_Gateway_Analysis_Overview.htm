SecureSpan™ Gateway Analysis Overview

 NOTE: The Status Bar under the [Services] tab contains a user, SecureSpan™ Gateway, and node information message when [View] > Status Bar is selected from the Main Menu (check box checked). See SecureSpan™ Manager Interfaces for more information. In the SecureSpan™ Manager, the term "identity" includes both users and groups, the term "user" can represent an individual human or machine, and the term "SecureSpan Gateway" includes all of the SecureSpan Gateway products.

The SecureSpan™ Manager uses two sources to monitor the following performance metrics in one or more SecureSpan™ Gateways:

  • SecureSpan Gateway cluster node statistics

  • Service message statistics

  • Service policy changes

  • SecureSpan Gateway connectivity and communication policies

  • General transaction traffic

  • Service-specific message processing event policies

  • Administrative event-specific auditing policies.

 

Use the Gateway Cluster Status window and Gateway Audit Events window to analyze and validate Service Level Agreement (SLA) compliance, plan network resources, troubleshoot connectivity issues, view detailed service-specific audit policy that represent service message processing and administration events, and monitor the general performance and activity of the SecureSpan Gateway.
