SecureSpan™ Gateway Analysis Overview

 NOTE: The Status Bar under the [Services] tab contains a user, SecureSpan™ Gateway, and node information message when [View] > Status Bar is selected from the Main Menu (check box checked). See SecureSpan™ Manager Interfaces for more information.

The SecureSpan™ Manager uses three sources to monitor the following performance metrics in one or more SecureSpan™ Gateways:

  • Gateway cluster node statistics

  • Service message statistics

  • Service policy changes

  • Gateway connectivity and communication messages

  • General transaction traffic

  • Service-specific message processing event messages

  • Administrative event-specific auditing messages.

 

 NOTE: Web services and XML applications are collectively referred to as "services."

 

Use the Gateway Cluster Status window and Gateway Log to analyze and validate Service Level Agreement (SLA) compliance, plan network resources, troubleshoot connectivity issues, and monitor the general performance and activity of the SecureSpan™ Solution. Use the Gateway Audit Messages window to view detailed service-specific audit messages that represent service message processing and administration events.
