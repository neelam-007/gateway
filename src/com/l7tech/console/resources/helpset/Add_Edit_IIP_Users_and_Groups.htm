Add and Edit Internal Identity Provider Users and Groups

 NOTE: The Internal Identity Provider (IIP) is configured during the installation and configuration of the SecureSpan™ Gateway. The IIP cannot be edited or deleted in the SecureSpan™ Manager.

In the SecureSpan™ Manager, the Internal Identity Provider (IIP) is used to manage and consume SecureSpan™ Gateway users and groups. The IIP is pre-configured with:

 

 NOTE: In the SecureSpan™ Manager, the term "identity" includes both users and groups, and the term "user" can represent an individual human or machine.

 

  • Two default access groups:

    • "Gateway Administrators" have access to the full-featured and privileged Administrator Interface, and  

    • "Gateway Operators" have access to the partially-featured read-only Operator Interface.  

  • A default user ("admin") that, as a member of the Gateway Administrators group, has access to the Administrator Interface.

 NOTE: See SecureSpan™ Manager Interfaces for more information about the Administrator Interface and Operator Interface.

 

All Manager users must be assigned to either the Gateway Administrators or Gateway Operators access groups. Users without a group assignment will not be able to access the Manager. Additional users and groups can be added to the IIP at any time.

 

 TIP: You can add users to groups from the Group Properties form or the User Properties form. To view the read-only properties of the Internal Identity Provider (IIP), double-click the IIP name in the [Identity Providers] tab, or right-click the IIP name and select Properties from the drop-down list.

Add a New Internal Identity Provider User

 NOTE: You cannot duplicate External Identity Provider (EIP) user credentials in the Internal Identity Provider. Make sure that the user information used in the SecureSpan™ Manager is unique. For more information on EIPs, see Add and Edit External Identity Providers.

In the SecureSpan™ Manager, a new user is added to the Internal Identity Provider (IIP) with the Create Internal User form (basic information) and the optional User Properties form.

 

  1. Access the Create Internal User form by one of three methods:

    • Click the "Create Internal User" link on the Home Page

 NOTE: The Home Page appears upon startup or when the Main Tool Bar's [Home] button is clicked.

    • Click [Tasks] > Create Internal User from the Main Menu

    • Right-click the IIP name in the [Identity Providers] tab. A drop-down menu appears. Select Create User.

  1. In the Create Internal User form:

    • Enter a user name into the User Name field

    • Enter a user password into the Password field

 NOTE: The minimum password size is six characters and the maximum password size is thirty-two characters.

    • Enter the same user password into the Confirm field.

  1. Do one of the following:

    • If you do not want to specify additional user information, then click [Create] without checking the Define Additional Properties check box. The user is added to the IIP. See Search, Edit, or Delete an Internal Identity Provider User or Group to view, edit, and/or delete the user at any time

 NOTE: New SecureSpan™ Manager users must be assigned to an access group to enable their login credentials.  

    • To specify the full name, email, authentication details, and group membership for the new user, proceed to Configure User Properties.

Configure User Properties

 NOTE: The properties in the User Properties form are recommended. If you click [Cancel] while configuring user properties, then the SecureSpan™ Manager will still save the information entered in the Create Internal User form.

 

  1. After you complete the fields in Create Internal User form, check the Define Additional Properties check box. Click [Create].

  1. The User Properties form appears. In the [General] tab:

    • Enter the user's first name into the First Name field

    • Enter the user's last name into the Last Name field

    • Enter the user's email address into the Email field.

 NOTE: The [Change Password] button is addressed in the Search, Edit, or Delete an Internal Identity Provider User or Group section.

  1. Click the [Membership] tab. The blank User Groups window appears. Do one of the following:

    • If you want to view or revoke the user's certificate without adding the user to one or more groups, proceed to Step #4

    • If you do not want to add the user to one or more groups or view or revoke the user's certificate, proceed to Step #5

    • If you want to add the user to one or more groups, click [Add]. The Add Users to the Group form appears. Add the user to one or more IIP groups as follows:

 NOTE: Use the instructions in Search, Edit, or Delete an Internal Identity Provider User or Group to modify group membership.

 NOTE: Assign new SecureSpan™ Manager users to one of the default access groups. The "Gateway Administrators" access group permits Administrator Interface access, and the "Gateway Operators" group permits Operator Interface access. For more information about the Manager interfaces, see SecureSpan™ Manager Interfaces.

Add to a Single Group

To add the user to a single group, select the group's name and click [Add]. The Add Users to the Group form closes and the group is added to the User Groups window in the User Properties form.

Add to Multiple Groups

      • Add the user to multiple groups by one of two methods:

        • To select a consecutive range of groups, press [Shift] on your keyboard, left-click the first group in the range, and then left-click the last group in the range

        • To select groups placed throughout the list, press [Ctrl] on your keyboard as you click each group name.

      • Click [Add]. The Add Users to the Group form closes and the groups are added to the User Groups window in the User Properties form.

 

  1. Click the [Certificate] tab. The Certificate Status window appears. Do the following:

View a Certificate

To view a certificate assigned to the user:

    • Double-click the certificate name in the Certificate Status window. An information window appears

    • View certificate information. Click [X] to close the window.

Revoke a Certificate

To revoke a certificate:

    • Select the certificate name in the Certificate Status window

    • Click [Revoke]. The certificate is removed from the Certificate Status window.

  1. In the User Properties form, click [OK]. The form closes and the user is added to the Internal Identity Provider.

 

 NOTE: Use the Search Identity Provider form to search the users belonging to the Internal Identity Provider. See Search, Edit, or Delete an Internal Identity Provider User or Group or Search Identity Providers for more information.

Add a New Internal Identity Provider Group

 NOTE: A group's SecureSpan™ Manager access rights are determined by the access group assignments of the individual users in the group. If you want the give the "SystemAdministrators" group Administrator Interface access, for example, then you must assign each user in that group to the "Gateway Administrators" access group.

In the SecureSpan™ Manager, a new group is added to the Internal Identity Provider (IIP) with the Create Internal Group form (basic information) and the optional Group Properties form.

 

  1. Access the Create Internal Group form by one of three methods:

    • Click the "Create Internal Group" link on the Home Page

 NOTE: The Home Page appears upon startup or when the Main Tool Bar's [Home] button is clicked.

    • Click [Tasks] > Create Internal Group from the Main Menu

    • Right-click the IIP name in the [Identity Providers] tab. A drop-down menu appears. Select Create Group.

  1. In the Create Internal Group form, enter a descriptive group name into the Group Name field, then do one of the following:

    • If you do not want to specify additional group information, then click [Create] without checking the Define Additional Properties check box. The group is added to the IIP. See Search, Edit, or Delete an Internal Identity Provider User or Group to view, edit, and/or delete the group at any time

    • To specify a group description and/or add users to the group, proceed to Configure Group Properties.

Configure Group Properties

 NOTE: The properties in the Group Properties form are recommended. If you click [Cancel] while configuring group properties, then the SecureSpan™ Manager will still save the information entered in the Create Internal Group form.

 

  1. After specifying the group name in the Create Internal Group form, check the Define Additional Properties check box. Click [Create].

  1. The Group Properties form appears. In the [General] tab, enter a descriptive assertion about the group into the Description field.

  2. Click the [Membership] tab. The blank Group Membership window appears. Do one of the following:

    • Proceed to Step #4 if you do not want to add users to the group

    • Click [Add] to add one or more users to the group. The Add Users to the Group form appears. Add one or more IIP users to the group as follows:

 NOTE: Use the instructions in Search, Edit, or Delete an Internal Identity Provider User or Group to modify group membership.

Add a Single User

To add a single user to the group, select the user's name and click [Add]. The Add Users to the Group form closes and the user is added to the Group Membership window in the Group Properties form.

Add Multiple Users

      • Add multiple users to the group by one of two methods:

        • To select a consecutive range of users, press [Shift] on your keyboard, left-click the first user in the range, and then left-click the last user in the range

        • To add users placed throughout the list, press [Ctrl] on your keyboard as you click each user name that you want to add.

      • Click [Add]. The Add Users to the Group form closes and the users are added to the Group Membership window in the Group Properties form.

 

  1. In the Group Properties form, click [OK]. The form closes and the group is added to the IIP.

 

 NOTE: Use the Search Identity Provider form to search the groups belonging to the Internal Identity Provider. See Search, Edit, or Delete an Internal Identity Provider User or Group or Search Identity Providers for more information.

Search, Edit, or Delete an Internal Identity Provider User or Group

 NOTE: You can only edit Internal Identity Provider (IIP) and Federated Identity Provider users and groups in the SecureSpan™ Manager. Use the appropriate management program to edit the users and groups that belong to an External Identity Provider. Use the following instructions to modify IIP group membership from the User Properties form or the Group Properties form. See View or Edit User or Group Properties for instructions.

In the SecureSpan™ Manager, Internal Identity Provider (IIP) users and groups are edited in two steps:

  • The Search Identity Provider form is used to find, and if necessary, delete the target user or group

  • The User Properties form or the Group Properties form is used to edit the properties of the respective user or group and to modify group membership.

 

  1. Access the Search Identity Provider form by one of two methods:

 TIP: The Search Identity Provider form can be used to search the contents of External Identity Providers, Federated Identity Providers, or the Internal Identity Provider. See Search Identity Providers for more information.

    • Click the "Search Identity Provider" link on the Home Page

 NOTE: The Home Page appears upon startup or when the Main Tool Bar's [Home] button is clicked.

    • Click [Tasks] > Search Identity Provider from the Main Menu

    • Right-click the IIP name in the [Identity Providers] tab. A drop-down menu appears. Select Search Identity Provider.

  1. In the Search Identity Provider form:

    • If not already selected, select the IIP name from the Search drop-down list

    • Select the search target — users, groups, or both — from the Type drop-down list

    • If necessary, specify any search criteria using the Name drop-down list and field

 TIP: The Search Identity Provider form supports the following wild card characters in the Name field:

  • The asterisk wild card (*) retrieves terms with a common initial or subsequent spelling. It matches any string of zero or more characters

  • The question mark wild card (?) substitutes one character in a term. Use within a word or at the end of a word.

    • Click [Search].

 NOTE: To stop a search that is in progress, click [Stop].

  1. Search results appear in the Search Results window. Do any of the following:

Delete a User or Group

 NOTE: An error message will appear if you try to delete a user or group that is being used by a security policy. To delete the user or group, you must first delete the identity from all of the policies that it is attached to. See Configure and Edit Security Policies for more information.

To delete a user or group:

 

    1. Select the target user or group name in the Search Results window.

 NOTE: At least one administrative user must be present in the SecureSpan™ Manager. An error message will appear if you try to delete all administrative users or the last administrative user.  

    1. Click [Delete]. A pop-up confirmation message appears. Click [OK].

    2. The user or group is deleted from the IIP. Do one of the following:

      • Make further changes as necessary, or

      • If further changes are not required, click [Close] to close the Search Identity Provider form.

View or Edit User or Group Properties

To modify a user:

 

    1. In the Search Results window in the Search Identity Provider form, double-click the target user name or select the user name and click [Select].

    2. The User Properties form appears:

      • To change the user's password, click [Change Password]. The Change Password form appears:

        1. Enter the new password into the New Password field.

        2. Repeat the new password in the ReType New Password field.

        3. Click [OK].

      • To modify the user properties or to add the user to one or more groups, follow the instructions in Configure User Properties

      • To remove the user from one or more groups:

 NOTE: Users must be assigned to the "Gateway Administrators" access group or "Gateway Operators" access group in order to use the SecureSpan™ Manager. When removing a user from one or more groups, ensure that the user remains a member of one of the access groups. For more information, see Configure User Properties.

        1. Click the [Membership] tab.

        2. The groups that the user belongs to are displayed in the User Groups window.

          • To remove the user from a single group, select the group name and click [Remove]

          • To remove the user from a consecutive range of groups, press [Shift] on your keyboard, left-click the first group in the range, and then left-click the last group in the range. Click [Remove]

          • To remove the user from multiple groups placed throughout the list, press [Ctrl] on your keyboard as you click each group name that you want to remove the user from. Click [Remove].

        3. The group(s) is/are removed from the User Groups window. Do one of the following:

          • Make further changes as necessary, or

          • If further changes are not required, click [OK] to close the User Properties form.

 

    1. If further user changes are not required, then click [Close] to close the Search Identity Provider form.

 

To modify a group:

 

    1. In the Search Results window in the Search Identity Provider form, double-click the target group name or select the group name and click [Select].

    2. The Group Properties form appears:

      • To modify the group properties or add new users to the group, follow the instructions in Configure Group Properties

      • To remove one or more users from the group:

        1. Click the [Membership] tab.

        2. The users that belong to the group are displayed in the Group Membership window.

          • To remove a single user, select the user name that you want to remove. Click [Remove]

          • To remove a consecutive range of users, press [Shift] on your keyboard, left-click the first user in the range, and then left-click the last user in the range. Click [Remove]

          • To remove users placed throughout the list, press [Ctrl] on your keyboard as you click each user name that you want to remove. Click [Remove].

        3. The user(s) is/are removed from the Group Membership window. Do one of the following:

          • Make further changes as necessary, or

          • If further changes are not required, click [OK] to close the Group Properties form.

 

    1. If further group changes are not required, then click [Close] to close the Search Identity Provider form.
