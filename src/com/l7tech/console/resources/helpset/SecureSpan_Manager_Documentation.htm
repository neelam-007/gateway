About this Help System

 NOTE: See SecureSpan™ Manager Interfaces for more information about the Administrator Interface and Operator Interface.

Two role-based versions of the SecureSpan™ Manager interface are available: a full-featured and privileged Administrator Interface for Administrators and a partial view-only Operator Interface for Operators. This full-version Help System includes all features and functions available in the Administrator Interface of the Manager. The documentation assumes a general knowledge of client operating systems, Internet terminology, HTTP security technologies, XML, and Web services. Although Operators have access to the full Help System, only particular sections will be relevant to the features that they are permitted to access. See SecureSpan™ Manager Operator Documentation for a list of the Help System sections relevant to the Operator Interface.

 

This Help System does not include installation instructions, add-on assertion installation instructions, SecureSpan™ Solution specifications, or external product information.

Help System Navigation

The SecureSpan™ Manager Help System is accessed by one of three methods:

  • Select [Help] > Help Topics from the Main Menu

  • Press the [F1] keyboard key, or

  • Click the [Help] button when available in a Manager form or wizard.

The Help System includes several features that allows for the easy access, navigation, and searchability of help information.  

Help System Tabs

Use the tabs in the left panel of the Help System window to view and search help information.

Table of Contents

The [Table of Contents] tab, accessed by clicking  (Table of Contents), organizes help information into expandable folders. Each page in the folder represents a help topic. When you double-click a closed folder, it opens to display its topic pages; when you double-click an open folder, it closes. Selected topics are viewed in the right panel.

Index

The [Index] tab, accessed by clicking  (Index), lists the keywords and keyword phrases associated with Help System topics. Click a keyword or phrase to open its associated topic page.

Search

The [Search] tab, accessed by clicking  (Search), allows you to search for specific words in the Help System:

  • Enter a search word or phrase in the Find field

  • Press [Enter] on your keyboard

  • Click the topic of interest in the Search pane. The Help System topic page appears in the right panel.

Navigation

The  (Previous) and  (Next) buttons in the upper Help System panel navigate you through the history of topics that you have visited.

Print Help Topics

Print information directly from the Help System window as follows:

  1. Highlight the target topic or folder.

  2. From the upper Help System panel, click  (Page Setup). The Page Setup form appears.

  3. Enter and select printing options. Click [OK] to return to the Help System.

  4. From the upper Help System panel, click  (Print). The Print form appears.

  5. Enter and select printing options. Click [OK].
