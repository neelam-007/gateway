Connecting and Disconnecting

Once the Manager preferences have been established with the Gateway server name and the correct port number, connecting is a very simple procedure. This can be accomplished through the easily accessible UI buttons or via the same options offered under the Session menu.

 

To connect to the Gateway via the Manager:

  1. Click the Connect button. The Login dialog box appears.

  2. Type the appropriate credentials in the Login and Password fields.

  3. Click Login. The Manager populates the UI screens with your Gateway content.

 

To disconnect from the Gateway via the Manager:

  1. Click the Disconnect button.

  2. The Manager removes all Gateway content and highlights the Connect button in the UI.

 

To exit from the Manager:

  1. Go to the Session menu.

  2. Select Exit. The Manager application closes down.
