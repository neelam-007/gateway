Setup Overview

Administrators must setup several components before being able to leverage the Gateway's functionality. Every organization will use the Manager differently. Administrators should be familiar with their corporate security standards and Web Service strategy.

The checklist below includes the key tasks that every administrator should complete when first setting up the Gateway. Before completing this list, administrators should decide on the scale of their Gateway implementation, their policy approach, and the future strategy for their corporate Web services.

Checklist:

¨  Set the user interface preferences

¨  Add external identity providers (optional)

¨  Create users and groups in the internal identity provider (optional)

¨  Publish Web services

¨  Create security policies for each Web Service
