Troubleshooting Mode

NOTE: The Linux version of the SecureSpan™ Bridge does not include a separate Troubleshooting Mode. Log information that is similar to the Microsoft Windows MS-DOS window below will appear whenever the "Bridge.sh" file in run in Linux.

The Microsoft Windows version of the SecureSpan™ Bridge interface contains two modes:

  • A GUI Mode used to configure and manage the SecureSpan Bridge functionality (see Interface), and

  • A Troubleshooting Mode used by Network Administrators to gather additional information about system errors.

 

Each mode is launched from a different shortcut icon in the program menu. To access the Troubleshooting Mode:

  1. From the Windows Desktop, click [Start] > All Programs > SecureSpan Bridge > SecureSpan Bridge in Troubleshooting Mode.

  1. An MS-DOS window populated with diagnostic information appears. If troubleshooting assistance is required, then copy and paste the information into an email. Send the email to a Technical Support Representative as outlined in Technical Support.

  1. Click the [X] in the top right corner to close the window.
