SecureSpan™ Overview

NOTE: SecureSpan™ supports the following standards: XML 1.0, SOAP 1.1, XPath 1.0, WSDL 1.1, LDAP 3.0, SAML 1.1, PKCS #10, X.509 v3 Certificates, W3C XML Signature 1.0, W3C XML Encryption 1.0, SSL 2.0 / 3.0, JMS 1.0, WS-Security 1.0, WS-Trust 1.0, XSLT 1.0, WS-SecureConversation, WS-Metadata Exchange, WS-Policy, WSRP, MQ Series, WS-I, and WS-I BSP.

Layer 7 Technologies' SecureSpan™ product line is composed of three interoperable products — the SecureSpan™ Gateway, the SecureSpan™ Bridge, and the SecureSpan™ Manager — that protect applications exposed as Web services, connect applications across security and identity domains, and validate policy compliance end-to-end across a transaction.

 

 

SecureSpan™ Deployment Architecture

 

SecureSpan™ Gateway

The SecureSpan™ Gateway is a policy-optimized and ASIC-accelerated XML Firewall and Web services Gateway that protects and controls how shared Web services are accessed by and exposed to external applications. In accordance with customer needs, the SecureSpan Gateway is delivered in one of four form factors:

  • As a software package running on general application servers or corporate-mandated hardware (XServer)

  • As a software package for scalable XML firewalling in the data center using IBM’s eServer™ BladeCenter (XBlade)

  • As a ASIC-accelerated DMZ appliance (XScreen), or

  • As a 64-bit ASIC-accelerated appliance for EAI or ESB implementations (XSpeed).

In this Help System, the term “SecureSpan Gateway” includes all of the SecureSpan Gateway products. For information about creating and managing SecureSpan Gateway Accounts in the SecureSpan™ Bridge, see Manage SecureSpan™ Gateway Accounts.

 

SecureSpan™ Bridge

NOTE: For program and release information, click [Help] > About from the Main Menu to open the About the SecureSpan Bridge form.

The SecureSpan™ Bridge is a cross-domain enablement product that speeds and secures Web services integrations that span identity and security domains. The SecureSpan Bridge documentation includes an Installation Manual, this program-based Help System, and an electronic (PDF) and printed User Manual based on the Help System content.

 

SecureSpan™ Manager

NOTE: The SecureSpan™ Manager and SecureSpan Manager documentation are required to configure identity bridging. See Identity Bridging for more information.

The SecureSpan™ Manager is a GUI-based application that allows SecureSpan Gateway administrators to centrally define, provision, verify, and audit fine-grained security and connectivity policies for cross-domain Web services and XML integrations. The SecureSpan Manager documentation includes an Installation Manual, a program-based Help System, and an electronic (PDF) and printed User Manual.
