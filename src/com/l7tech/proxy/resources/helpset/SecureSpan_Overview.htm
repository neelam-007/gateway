SecureSpan™ Overview

 TIP: See www.layer7tech.com for more information about the SecureSpan™ products.

The SecureSpan™ product line enables the simple creation and management of Web service and XML application policies. Three separate yet interoperable products combine to form a powerful security solution: the SecureSpan™ Gateway, the SecureSpan™ Manager, and the SecureSpan™ Bridge.

 

 NOTE: SecureSpan™ supports the following standards: XML 1.0, SOAP 1.1, XPath 1.0, WSDL 1.1, LDAP 3.0, SAML 1.1, PKCS #10, X.509 v3 Certificates, W3C XML Signature 1.0, W3C XML Encryption 1.0, SSL 2.0 / 3.0, JMS 1.0, WS-Security 1.0, WS-Trust 1.0, XSLT 1.0, WS-SecureConversation, WS-Metadata Exchange, WS-Policy, WSRP, MQ Series, WS-I, and WS-I BSP. Additional supported standards and tool kits are referenced with their applicable feature or function.

SecureSpan™ Gateway

 NOTE: For brevity, the term “SecureSpan™ Gateway” includes all of the SecureSpan Gateway products: SecureSpan™ XServer™, SecureSpan™ XBlade™, SecureSpan™ XScreen™, and SecureSpan™ XSpeed™. For more information about the SecureSpan Gateway products, see www.layer7tech.com.

The SecureSpan™ Gateway is the centerpiece of SecureSpan’s™ threat protection capabilities. The SecureSpan Gateway performs various XML application and services firewalling and threat protection functions, including access management, privacy enforcement, data validation, routing, transformation, and auditing. In accordance with specific customer needs, the SecureSpan Gateway is deployed in one of the following form factors :

  • SecureSpan™ XServer™: Software package for XML firewalling and Web services access control running on general application servers or corporate-mandated hardware. Available with the optional XML ASIC accelerator

  • SecureSpan™ XBlade™: Software package for scalable XML firewalling in the data center using IBM’s eServer™ BladeCenter architecture

  • SecureSpan™ XScreen™: High-performance appliance for accelerated threat protection and XML firewalling in the DMZ. Incorporates ASIC-accelerated XML parsing, inspection, data validation, and pattern detection. Can be clustered to meet specific throughput requirements

  • SecureSpan™ XSpeed™: First ASIC accelerated 64-bit Web services Gateway for generalized XML processing at the edge of EAI or ESB implementations. Can be clustered to meet specific throughput requirements.

 

In this Help System, the term “SecureSpan Gateway” includes all of the SecureSpan Gateway products. For information about creating and managing SecureSpan Gateway Accounts in the SecureSpan™ Bridge, see SecureSpan™ Gateway Accounts Overview.

 

SecureSpan™ Manager

 NOTE: The SecureSpan™ Manager and "SecureSpan™ Manager User Manual" are required to configure identity bridging. See Identity Bridging Overview for more information.

The SecureSpan™ Manager is a GUI-based application that allows administrators to centrally define, manage, verify, and audit fine-grained policies for Web services and XML applications. Through a declarative policy language of assertions, users create or connect to shared services, establish trust and identity sources using existing infrastructure, and use these sources to define policies. The SecureSpan Manager decouples policy control from both the provider-side and client-side business logic, transforming policy management and service governance from a complex programming challenge into a simple administrative task. For more information about the SecureSpan Manager, see the “SecureSpan™ Manager User Manual” or SecureSpan Manager-based Help System.

 

SecureSpan™ Bridge

The SecureSpan™ Bridge is a powerful client-side solution for streamlining and scaling integrations that span domains. The SecureSpan Bridge removes the programming burden and multi-platform issues associated with client applications, off-loading a broad range of security operations such as the provisioning and management of PKI, identity bridging between security domains, non-programmatic SAML generation, SSO extension, replay attack prevention, end-to-end QoS, and non-repudiation. For more information about the SecureSpan Bridge, see SecureSpan™ Bridge Overview.
