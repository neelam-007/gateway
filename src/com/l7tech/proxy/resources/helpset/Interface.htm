Interface

NOTE: The Troubleshooting Mode (see Troubleshooting Mode) of the SecureSpan™ Bridge is launched separately from the general user interface.

The SecureSpan™ Bridge general user interface is divided into three main areas:

 

The SecureSpan™ Bridge GUI

 

Main Menu

TIP: Many Main Menu items can be accessed through keyboard shortcuts where provided. Press [Alt] on your keyboard to enable the shortcuts. The initiating letter for a shortcut is underlined in the Main Menu and in each respective drop-down menu.

Contains drop-down menus that include all of the program features, functions, and information.

Main Tool Bar

Contains shortcut buttons to commonly used program features.

Account List

Contains a list of the configured SecureSpan™ Gateway Accounts, called “Gateway Accounts” for short. Each column in the Account List corresponds with a property of the associated Gateway Account. See Manage SecureSpan™ Gateway Accounts for more information.
