Analyze SecureSpan™ Bridge Overview

 NOTE: The term "identity" includes both users and groups, the term "user" can represent an individual human or machine, the term “service” includes both Web services and XML applications, and the term "SecureSpan™ Gateway" includes all of the SecureSpan Gateway products.

Two sources of SecureSpan™ Bridge performance information are available:

  • A GUI window that displays session-based request and response message activity, and

  • Two local Log files that display connectivity and communication messages for the client application and one or more SecureSpan™ Gateways.

 

 TIP: You can get detailed performance metrics for one or more SecureSpan™ Gateways through the SecureSpan™ Manager. See the SecureSpan™ Gateway Analysis Overview section in the “SecureSpan™ Manager User Manual” or program-based “SecureSpan™ Manager Help System.”
