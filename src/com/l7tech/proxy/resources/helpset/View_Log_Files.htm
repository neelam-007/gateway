View Log Files

During the first installation of the SecureSpan™ Bridge, a “logging.properties” file is installed that will govern what information is included in the default SecureSpan Bridge Log files. The “logging.properties” file should not be changed unless recommended by your Network Administrator or Layer 7 Technologies for troubleshooting purposes.

 

Navigate to the local location of the SecureSpan Bridge Log files to view detailed connectivity and communication messages:

 

Windows Location

If the SecureSpan™ Bridge was installed in a Microsoft Windows XP environment, then the Log files will be located in the “C:\Documents and Settings\<user>\.l7tech” directory.

 

Linux Location

If the SecureSpan™ Bridge was installed in a Linux environment, then the Log files will be located in the user home or “/root/.l7tech” directory.

 

The SecureSpan Bridge Log files are named “ssa0.log” and “ssa1.log,” respectively. Information is logged to the two Log files in a pre-configured rollover fashion. The SecureSpan Bridge logs information to the “ssa0.log” file until it reaches the maximum file size of 10MB; the SecureSpan Bridge then automatically switches and starts logging information to the “ssa1.log” file. When the “ssa1.log” file reaches the maximum file size of 10MB, the SecureSpan Bridge will transition back to logging information to the “ssa0.log,” automatically overwriting the older log information in the “ssa0.log” file.
