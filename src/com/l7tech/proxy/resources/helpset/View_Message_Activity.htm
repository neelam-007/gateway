View Message Activity

The SecureSpan™ Bridge’s Recent Message Traffic window retains detailed information about the last 32 request and response messages processed in the current SecureSpan Bridge session. To access the Recent Message Traffic window:

 

  1. Select [Window] > Recent Message Traffic from the Main Menu.

  2. The Recent Message Traffic window appears in a separate window. View, hide, format, and clear the request and response messages as follows:

 NOTE: The messages in the Recent Message Traffic window are only for the current SecureSpan™ Bridge session. Restarting the SecureSpan Bridge will clear the Recent Message Traffic window. The Recent Message Traffic window can also be cleared without closing the SecureSpan Bridge. See Clear Messages below for information.

View Message Details

 NOTE: The messages in the Recent Message Traffic window are presented in chronological order, from oldest down to newest according to when they were processed by the SecureSpan™ Bridge.

To view the details of a message in the Recent Message Traffic window, select the target message name and time of receipt from the left window panel. Message details will appear in the right panel as formatted XML with the complete HTTP headers for the message. The messages in the Recent Message Traffic window are undecorated (without the required security) and saved by the SecureSpan™ Bridge upon receipt from the client application (request messages) or just before being sent back to the client application (response messages).

Reformat Message XML

To reformat the XML messages in the right panel of the Recent Message Traffic window, check the Reformat XML check box under the left panel. When enabled, the displayed XML messages are reformatted and re-indented, making them easier to read.

 

 TIP: Before copying a digitally signed message from the Recent Message Traffic window to paste elsewhere, uncheck the Reformat XML check box. Changing the formatting of a message may invalidate the message signature.

 

Clear Messages

To clear all messages from the Recent Message Traffic window, click the [Clear Recent Messages] button under the right panel. All messages are immediately removed.

  1. To close the Recent Message Traffic window, click the [Close] button under the right panel.
